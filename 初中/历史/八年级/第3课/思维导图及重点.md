# 第3课 社会主义制度的建立——思维导图及重点

## 一、思维导图 (树状图)

第3课 社会主义制度的建立
├── 过渡时期总路线 (1953年)
│   └── 核心内容：“一化三改” (工业化 + 三大改造)

    └── 性质：第一部社会主义类型的宪法

## 二、本课核心重点 (必须记住)

1. **“一五计划”的主要目标**：集中主要力量发展**重工业**。
2. **“一五计划”的标志性成就**：长春第一汽车制造厂、武汉长江大桥（1957年）。
3. **三大改造完成的意义**：标志着**社会主义基本制度**在我国建立起来，我国从此进入**社会主义初级阶段**。
4. **资本主义工商业改造的独创性方式**：**赎买政策**（实现了和平过渡）。
5. **1954年宪法的性质**：我国第一部**社会主义类型**的宪法，真正反映人民利益。
