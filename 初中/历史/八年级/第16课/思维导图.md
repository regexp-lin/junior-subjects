# 第16课 独立自主的和平外交 - 思维导图（树状图）

*   **第16课 独立自主的和平外交**
    *   **一、独立自主的和平外交政策**
        *   确立：新中国成立时毛泽东宣告
        *   内容：愿与遵守平等、互利及互相尊重领土主权等原则的国家建交
        *   面临的困境：美国封锁禁运、军事威胁，西方国家不承认新中国
    *   **二、和平共处五项原则**
        *   提出：1953年底周恩来接见印度代表团时首次提出
        *   完善：1954年与缅甸、印度发表联合声明共同倡导
        *   内容
            *   互相尊重领土主权
            *   互不侵犯
            *   互不干涉内政
            *   平等互利（原为平等互惠）
            *   和平共处
        *   意义：成为处理国际关系的基本准则，在国际上产生深远影响
    *   **三、万隆会议**
        *   时间、地点：1955年，印度尼西亚万隆
        *   性质：第一次没有西方殖民国家参加的亚非会议
        *   议题：亚非国家共同关心的国际问题与民族解放问题
        *   面临困难：各国社会制度不同、国家利益差异大
        *   周恩来的贡献："求同存异"方针
            *   促进会议圆满成功
            *   加强中国与亚非各国团结合作
            *   为许多国家与中国建交创造条件
