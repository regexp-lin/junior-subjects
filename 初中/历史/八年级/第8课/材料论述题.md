# 第8课 经济体制改革 - 材料论述题及答案要点

**1. 材料：** "我们分田到户，每户户主签字盖章，如以后能干，每户保证完成每户的全年上交公粮，不再向国家伸手要钱要粮。如不成，我们干部坐牢杀头也甘心。"——1978年安徽凤阳小岗村18户农民"生死状"

*   **题目：** 根据材料，结合所学知识，说明小岗村农民的做法在当时有何重大意义？
*   **答案要点：**
    *   小岗村农民冲破人民公社体制束缚，实行"分田包产到户、自负盈亏"
    *   成为中国农村改革的突破口和发源地
    *   开创了家庭联产承包责任制的先河，为全国农村改革提供了范例
    *   体现了人民群众的首创精神和改革勇气

**2. 材料：** "交够国家的，留足集体的，剩下都是自己的。"——家庭联产承包责任制分配原则

*   **题目：** 这一分配原则如何体现了农村改革对农民积极性的调动？
*   **答案要点：**
    *   明确了国家、集体、个人三者的利益分配关系
    *   农民在完成国家任务和集体提留后，剩余产品归自己所有
    *   实现了多劳多得，打破了"大锅饭"平均主义
    *   将农民利益与劳动成果直接挂钩，极大地激发了生产热情

**3. 材料：** 1984年10月，中共十二届三中全会通过《中共中央关于经济体制改革的决定》，明确指出："增强企业的活力，特别是增强全民所有制大中型企业的活力，是以城市为重点的整个经济体制改革的中心环节。"

*   **题目：** 根据材料，说明城市经济体制改革为什么以增强企业活力为中心环节？
*   **答案要点：**
    *   国有企业是国民经济的主导力量，其活力直接影响经济发展
    *   计划经济体制下企业缺乏自主权，束缚了生产积极性
    *   增强企业活力可以解放生产力，提高经济效益
    *   通过扩大自主权、实行经营责任制等手段激发企业创造力

**4. 材料：** "我国经济体制改革确定什么样的目标模式，是关系整个社会主义现代化建设全局的一个重大问题。实践的发展和认识的深化，要求我们明确提出，我国经济体制改革的目标是建立社会主义市场经济体制。"——1992年中共十四大报告

*   **题目：** 根据材料，说明建立社会主义市场经济体制有何重大意义？
*   **答案要点：**
    *   明确了经济体制改革的方向和目标，是关系现代化建设全局的重大决策
    *   社会主义市场经济体制使市场对资源配置起基础性作用，提高经济效率
    *   有利于实现经济协调发展和稳定高速增长
    *   对推动社会主义现代化建设具有巨大作用

**5. 材料：** "改革开放从农村破题。大包干是农民的一大创举，搞家庭联产承包，把集体经济和农户的积极性都调动起来了。"——邓小平

*   **题目：** 结合材料，分析农村经济体制改革取得成功的原因。
*   **答案要点：**
    *   符合当时农村生产力发展水平和农民意愿
    *   家庭联产承包责任制调动了农民个体和集体的双重积极性
    *   土地集体所有制性质不变，坚持了社会主义方向
    *   党中央及时总结群众经验并推广，政策支持到位
