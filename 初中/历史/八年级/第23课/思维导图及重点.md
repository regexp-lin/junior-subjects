# 第23课 改革开放到新时代综合复习 - 思维导图及重点

## 一、 思维导图（树状图）

*   **第23课 改革开放到新时代综合复习**
    *   **一、历史转折**：十一届三中全会、改革开放
    *   **二、经济改革**：农村承包制、国企改革、市场经济
    *   **三、对外开放**：经济特区、全方位开放格局
    *   **四、理论发展**：邓小平理论、三个代表、科学发展观
    *   **五、新时代**：中国梦、四个全面、新发展理念
    *   **六、综合成就**：统一、外交、科技、民生

## 二、 重点记住的要点

1. **十一届三中全会**：1978年12月，**工作中心**转移到**经济建设**，实行**改革开放**，开启社会主义现代化建设**新时期**。

2. **经济体制改革**：农村**家庭联产承包责任制**（小岗村突破）；1992年十四大确立**社会主义市场经济体制**目标。

3. **对外开放格局**：**经济特区**（深圳等）→沿海城市→内地，形成**全方位、多层次、宽领域**开放格局。

4. **中国特色社会主义**：**邓小平理论**（十五大确立）、**三个代表**（十六大）、**科学发展观**（十七大）；基本路线**"一个中心、两个基本点"**。

5. **新时代战略**：**中国梦**（国家富强、民族振兴、人民幸福）；**四个全面**；**新发展理念**（创新、协调、绿色、开放、共享）；**"一带一路"**。
