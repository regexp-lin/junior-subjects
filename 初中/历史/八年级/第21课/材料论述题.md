# 第21课 活动课——让我们共同来感受历史 - 材料论述题及答案要点

**1. 材料：** 历史不是冰冷的过去，而是有温度的记忆。当我们听爷爷奶奶讲述他们年轻时的故事，看一张张发黄的老照片，触摸一件件锈蚀的旧物件，历史便鲜活起来。
*   **题目：** 材料说明了感受历史的哪些特点？我们应如何让历史"鲜活"起来？
*   **答案要点：**
    *   特点：历史可以是有温度、可触摸的；口述、实物等能够让历史更生动；普通人的故事也是历史的重要组成部分。
    *   方法：采访亲历者，记录口述史；收集老照片、老物件；参观博物馆、纪念馆；将个人、家庭故事与时代大背景联系起来。

**2. 材料：** 某中学开展"寻访家乡变迁"主题活动，同学们通过采访老人、查阅地方志、参观老建筑，了解了家乡从新中国成立到改革开放以来的巨大变化。
*   **题目：** 这种学习历史的方式有什么优点？对我们学习八下历史有什么启发？
*   **答案要点：**
    *   优点：贴近生活，易于理解；运用多种史料（口述、文献、实物）；将课本知识与实际结合；培养实践能力和史料实证素养。
    *   启发：学习历史要结合身边实际；可以开展类似活动了解改革开放、社会生活变迁等内容；多种方式结合能更全面地理解历史。

**3. 材料：** "以铜为镜，可以正衣冠；以史为镜，可以知兴替；以人为镜，可以明得失。"——唐太宗
*   **题目：** 结合所学知识，谈谈"以史为镜，可以知兴替"的含义，以及学习新中国史对我们有哪些启示。
*   **答案要点：**
    *   含义：通过学习历史可以了解朝代兴衰、社会变革的规律，为现实发展提供借鉴。
    *   启示：新中国的成就离不开正确的道路选择（社会主义、改革开放）；发展要立足国情；以人民为中心；坚持改革开放；汲取历史教训，避免走弯路。

**4. 材料：** 一件抗美援朝时期的老军装，一张1977年恢复高考时的准考证，一台20世纪80年代的黑白电视机……这些老物件静静地诉说着那个时代的故事。
*   **题目：** 实物史料在历史学习中有什么作用？请结合八下所学内容，说明这些老物件分别反映了什么历史。
*   **答案要点：**
    *   作用：直观反映历史；增强真实感和感染力；弥补文字史料的不足；有助于理解当时的社会生活。
    *   对应：军装——抗美援朝、保家卫国；准考证——恢复高考、尊重知识人才；黑白电视机——改革开放初期、人民生活改善。

**5. 材料：** 历史是最好的教科书，也是最好的清醒剂。学习党史、新中国史、改革开放史、社会主义发展史，是坚持和发展中国特色社会主义、把党和国家各项事业继续推向前进的必修课。
*   **题目：** 为什么说学习"四史"是必修课？作为青少年，我们应该如何传承历史、担当使命？
*   **答案要点：**
    *   原因：了解党和国家的奋斗历程；汲取经验教训；增强道路自信、理论自信、制度自信、文化自信；不忘初心、牢记使命。
    *   担当：认真学习历史，树立正确历史观；传承红色基因和爱国精神；珍惜今日幸福生活；努力学习，为实现中国梦贡献力量；做有理想、有本领、有担当的时代新人。
