# 第20课 活动课——生活环境的巨大变化 - 材料论述题及答案要点

**1. 材料：** 改革开放初期，我国城镇化率不到20%。到2020年，常住人口城镇化率超过60%，数以亿计的农村人口走进城市，分享了城镇化带来的发展成果。
*   **题目：** 材料反映了我国城镇化的什么变化？城镇化发展有何意义？
*   **答案要点：**
    *   变化：城镇化率从不到20%提高到超过60%；大量农村人口进入城市。
    *   意义：促进了城乡协调发展；改善了农民生活；推动了工业化、现代化；扩大了内需，促进了经济增长。

**2. 材料：** "我们既要绿水青山，也要金山银山。宁要绿水青山，不要金山银山，而且绿水青山就是金山银山。"——习近平
*   **题目：** 这段话体现了什么发展理念？结合所学知识，说明这一理念对我国发展的指导意义。
*   **答案要点：**
    *   体现了绿色发展、生态文明理念，强调经济发展与生态环境保护并重。
    *   意义：纠正了过去片面追求GDP的做法；推动产业结构转型升级；实现可持续发展；建设美丽中国；满足人民对美好生态环境的需求。

**3. 材料：** 近年来，我国实施大气污染防治行动计划，重点治理雾霾。许多城市空气质量明显改善，蓝天白云的天数增多，人民群众的获得感增强。
*   **题目：** 根据材料，说明我国在环境治理方面采取了什么措施？取得了什么成效？
*   **答案要点：**
    *   措施：实施大气污染防治行动计划；重点治理雾霾；加大环保投入。
    *   成效：城市空气质量明显改善；蓝天白云天数增多；人民群众获得感和幸福感增强；体现了以人民为中心的发展思想。

**4. 材料：** 从"脏乱差"的棚户区到整洁的现代化小区，从"出门一脚泥"到"出门上公交"，中国人的居住和出行环境发生了翻天覆地的变化。
*   **题目：** 材料反映了生活环境哪些方面的变化？这种变化说明了什么？
*   **答案要点：**
    *   变化：居住环境从棚户区到现代化小区；出行从泥路到公共交通便利。
    *   说明：改革开放以来人民生活环境显著改善；城镇化建设和基础设施建设取得巨大成就；体现了发展成果惠及人民；中国特色社会主义制度优势明显。

**5. 材料：** 建设美丽中国，是中华民族永续发展的千年大计。我们要坚持人与自然和谐共生，像对待生命一样对待生态环境。
*   **题目：** 结合材料，谈谈你对"美丽中国"建设的认识，以及青少年应如何参与生态文明建设。
*   **答案要点：**
    *   认识：美丽中国是中国梦的重要内容；关乎中华民族永续发展；要坚持人与自然和谐共生；是"五位一体"总体布局的重要组成部分。
    *   青少年参与：树立环保意识，节约资源；践行绿色出行、垃圾分类；参与环保宣传和志愿活动；从身边小事做起，为建设美丽中国贡献力量。
