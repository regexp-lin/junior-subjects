# 第18课 科技文化成就 - 材料论述题及答案要点

**1. 材料：** 如果六十年代以来中国没有原子弹、氢弹，没有发射卫星，中国就不能叫有重要影响的大国，就没有现在这样的国际地位。——邓小平
*   **题目：** 根据材料，说明"两弹一星"对我国国际地位有何影响？列举"两弹一星"的主要成就。
*   **答案要点：**
    *   影响：打破了超级大国的核垄断；大大提高了中国的国际地位；使中国成为有重要影响的大国。
    *   主要成就：1964年原子弹、1967年氢弹、1966年核导弹、1970年"东方红一号"人造卫星。

**2. 材料：** 1970年4月24日，中国成功发射了第一颗人造地球卫星。卫星上的《东方红》乐曲响彻太空。
*   **题目：** 材料中的卫星叫什么？这一成就有什么历史意义？
*   **答案要点：**
    *   名称："东方红一号"。
    *   意义：中国成为世界上第五个能独立发射人造地球卫星的国家；标志着中国航天事业的开端；提高了中国的国际地位。

**3. 材料：** 籼型杂交水稻的培育成功，是中国人民的骄傲，也是世界的骄傲。我的梦想是让杂交水稻覆盖全球。——袁隆平
*   **题目：** 袁隆平在农业科技领域作出了什么重大贡献？他的贡献有什么世界性意义？
*   **答案要点：**
    *   贡献：20世纪70年代成功培育籼型杂交水稻，被誉为"杂交水稻之父"。
    *   意义：为解决中国吃饭问题作出巨大贡献；对解决世界性饥饿问题具有重要意义；杂交水稻技术推广到许多国家。

**4. 材料：** 青蒿素的发现，是传统中医药献给世界的礼物。——屠呦呦获奖感言
*   **题目：** 屠呦呦因什么成就获得诺贝尔奖？这一成就对人类有何贡献？
*   **答案要点：**
    *   成就：发现青蒿素，开创治疗疟疾的新方法。
    *   贡献：使全球数亿疟疾患者受益；是中医药对世界的重大贡献；2015年屠呦呦成为首位获得科学类诺贝尔奖的中国科学家。

**5. 材料：** 2003年10月15日，中国首位航天员杨利伟乘坐神舟五号载人飞船进入太空，在轨飞行21小时后安全返回地面。
*   **题目：** 杨利伟的太空之旅标志着什么？中国载人航天事业的发展说明了什么？
*   **答案要点：**
    *   标志：中国成为世界上第三个独立掌握载人航天技术的国家；中国航天事业取得重大突破。
    *   说明：中国科技实力不断增强；综合国力显著提升；体现了自力更生、勇于登攀的"两弹一星"精神。
