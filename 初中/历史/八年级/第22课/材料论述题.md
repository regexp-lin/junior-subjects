# 第22课 新中国成立到改革开放综合复习 - 材料论述题及答案要点

**1. 材料：** "中华人民共和国中央人民政府今天成立了！"——1949年10月1日，毛泽东在天安门城楼庄严宣告
*   **题目：** 新中国成立有什么历史意义？为巩固政权，新中国随即采取了哪些措施？
*   **答案要点：**
    *   意义：开辟中国历史新纪元；推翻帝国主义、封建主义、官僚资本主义；中国成为独立自主的国家；壮大世界和平民主和社会主义力量。
    *   措施：西藏和平解放；抗美援朝；土地改革；建立人民代表大会制度；实施第一个五年计划。

**2. 材料：** 1956年底，我国基本完成了对农业、手工业和资本主义工商业的社会主义改造。社会主义制度在中国基本建立起来。
*   **题目：** 三大改造的内容和意义是什么？它与土地改革有什么不同？
*   **答案要点：**
    *   内容和意义：对农业、手工业进行合作化改造，对资本主义工商业实行公私合营；标志着社会主义制度在我国基本建立；生产资料私有制转变为公有制。
    *   区别：土地改革是废除封建土地所有制，实行农民土地所有制（私有）；三大改造是将私有制改造为公有制，确立社会主义制度。

**3. 材料：** 1958年，各地竞相放"卫星"，虚报粮食产量。有的地方提出"人有多大胆，地有多大产"。全民大炼钢铁，许多家庭把铁锅、铁门都拿去炼钢。
*   **题目：** 材料反映了什么历史现象？这种现象造成了什么后果？给我们什么教训？
*   **答案要点：**
    *   现象："大跃进"运动中的高指标、瞎指挥、浮夸风；全民大炼钢铁。
    *   后果：国民经济比例严重失调；资源浪费；人民生活困难。
    *   教训：经济建设必须尊重客观规律；不能急于求成；要坚持实事求是；浮夸和蛮干危害巨大。

**4. 材料：** 1964年10月16日，中国第一颗原子弹爆炸成功。邓稼先隐姓埋名数十载，为"两弹一星"事业奉献一生，被誉为"两弹元勋"。
*   **题目：** 我国"两弹一星"事业取得了哪些成就？有什么重要意义？
*   **答案要点：**
    *   成就：1964年原子弹；1967年氢弹；1970年"东方红一号"人造卫星。
    *   意义：打破了超级大国的核垄断；提高了我国国际地位和国防实力；增强了民族自信心；为后来航天事业奠定了基础。

**5. 材料：** "文化大革命"期间，民主与法制遭到严重践踏，大批干部和群众受到迫害。1976年10月，中共中央一举粉碎"四人帮"，结束了这场内乱。
*   **题目：** "文化大革命"有什么危害？粉碎"四人帮"有什么意义？这段历史对后来的改革开放有何影响？
*   **答案要点：**
    *   危害：民主法制被践踏；大批干部群众遭迫害；国民经济受严重影响；社会秩序混乱。
    *   意义：结束了十年内乱；为国家走上正常发展轨道创造条件。
    *   影响：深刻教训促使人们反思；认识到必须改革开放、以经济建设为中心；为十一届三中全会召开创造了条件。
