# 八年级上册地理 第三章《中国的自然资源》思维导图（树状图）

- 第三章 中国的自然资源
  - 第一节 中国的土地资源
    - 基本国情
      - 总量丰富
      - 人均占有量少
    - 土地利用类型
      - 耕地
      - 林地
      - 草地
      - 建设用地
    - 分布特点
      - 耕地主要在东部季风区平原和盆地
      - 林地多分布在山区
      - 草地多分布在西部内陆
    - 保护与利用
      - 节约用地
      - 合理利用
      - 保护耕地
  - 第二节 中国的水资源
    - 基本特点
      - 总量较丰富
      - 人均占有量少
      - 时空分布不均
    - 分布规律
      - 南多北少
      - 夏秋多、冬春少
    - 解决措施
      - 节约用水
      - 防治水污染
      - 跨流域调水
  - 第三节 中国的矿产资源
    - 特点
      - 种类齐全
      - 总量丰富
      - 分布广泛
    - 重要矿产
      - 煤
      - 石油
      - 铁矿
      - 有色金属
    - 开发利用
      - 合理开发
      - 综合利用
      - 保护资源与环境
