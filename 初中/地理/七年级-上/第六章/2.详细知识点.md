# 七年级上册地理 第六章《世界的发展差异》详细知识点

## 一、第一节 发展中国家与发达国家

### 1. 发展差异的主要表现
1. 经济方面：人均收入、产业结构、工业化水平存在显著差异。
2. 科技教育方面：研发投入、教育普及程度、人才储备不同。
3. 社会生活方面：医疗卫生、基础设施、社会保障水平不同。
4. 资源利用方面：资源开发能力和环境治理能力不同。

### 2. 国家类型划分
1. 发达国家：经济和科技较发达，第三产业比重高，劳动生产率较高。
2. 发展中国家：总体发展水平相对较低，但内部差异很大，部分国家发展迅速。

### 3. 空间分布特点
1. 发达国家主要分布在欧洲、北美洲和大洋洲部分地区。
2. 发展中国家主要分布在亚洲、非洲和拉丁美洲。
3. “南北差异”是历史与现实多种因素共同作用的结果。

### 4. 差异形成原因（理解）
1. 历史基础不同（工业化起步时间等）。
2. 资源条件和地理区位差异。
3. 科技与教育发展水平不同。
4. 国际分工和贸易体系影响。

## 二、第二节 国际合作

### 1. 为什么需要国际合作
1. 资源、环境、气候、疾病传播等问题具有跨国性。
2. 各国发展水平不同，优势互补可以提高整体效率。
3. 共同应对全球风险需要制度化合作。

### 2. 国际合作主要领域
1. 经济合作：贸易、投资、产业协作。
2. 科技合作：技术交流、联合研发。
3. 环境合作：减排、生态保护、污染治理。
4. 公共卫生合作：疫情监测、药物与疫苗协同。

### 3. 国际组织作用
1. 联合国等国际组织在协调国际关系、推动合作方面发挥重要作用。
2. 区域组织可推动区域内贸易、交通、能源和安全合作。

### 4. 合作原则
1. 平等互利。
2. 尊重差异。
3. 共同但有区别的责任（环境治理常见原则）。

## 三、本章高频考点
1. 发达国家与发展中国家的主要区别维度。
2. 世界发展差异的空间分布特征。
3. 全球性问题为何必须国际合作。

## 四、易错点
1. 把“发展中国家”与“落后国家”画等号。
2. 认为国际合作只在经济领域。
3. 忽视发展差异是动态变化的过程。
