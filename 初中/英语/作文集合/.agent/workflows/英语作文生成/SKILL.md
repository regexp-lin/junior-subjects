# 英语作文辅导

给定一篇英语作文题目，从多维度进行专业分析，在 **作文题目所在的同级目录下** 生成以下三个文件：

| 序号 | 文件名 | 内容 |
|------|--------|------|
| 1 | `作文要点.md` | 作文的核心要点，每个要点 1–2 句话总结 |
| 2 | `写作思路.md` | 如何思考这篇英语作文，从审题到下笔的完整思维过程 |
| 3 | `完整范文.md` | 一篇完整的高质量英语范文，附中文点评 |

适用于初中和高中阶段。

## 学段要求

| 学段 | 字数要求 | 词汇水平 | 语法要求 | 常见文体 |
|------|---------|---------|---------|---------|
| **初中** | 100 词左右 | 课标词汇为主，简单高频 | 简单句 + 少量并列句/复合句 | 书信、日记、介绍、看图写话、通知 |
| **高中** | 100–150 词 | 高级词汇替换、多样化表达 | 复合句、非谓语、倒装、强调句 | 书信/邮件、通知、演讲稿、议论文、读后续写 |

> 如用户未指定学段，默认按 **初中** 要求处理。可以在分析前主动询问用户学段。

## 英语作文评分维度参考

| 维度 | 说明 | 权重 |
|------|------|------|
| **内容完整** | 是否覆盖所有要点，信息无遗漏 | ★★★★★ |
| **语言准确** | 语法、拼写、标点是否正确 | ★★★★★ |
| **篇章结构** | 分段合理、逻辑通顺、衔接自然 | ★★★★ |
| **词汇丰富** | 是否使用多样化的词汇和短语 | ★★★★ |
| **句式多样** | 是否使用不同的句型结构 | ★★★ |
| **书写规范** | 大小写、标点、格式是否正确 | ★★★ |

## Workflow

当用户给出一个英语作文题目后，按以下步骤 **依次** 执行。

### 1. 确定输出目录

- 如果用户提供了作文题目所在的文件路径，则在该文件的 **同级目录** 下生成三个文件
- 如果用户只是口头给出题目，则询问用户希望将文件保存在哪个目录
- 如果目录下已存在同名文件，询问用户是否覆盖

---

### 2. 审题分析（内部步骤，不单独输出文件）

对题目进行全面审题，作为后续生成三个文件的基础：

#### 2.1 识别题型

判断作文类型并确定应对策略：

| 题型 | 特征 | 应对策略 |
|------|------|---------|
| 书信/邮件 | Dear..., Yours... | 注意格式（称呼、正文、结尾敬语、签名），语气得体 |
| 通知/公告 | Notice / Announcement | 开头说明目的，信息要素完整（时间、地点、内容） |
| 日记 | 给定日期和事件 | 用过去时为主，注意日期格式 |
| 看图写话 | 提供图片/图表 | 准确描述图片信息，适当发挥 |
| 话题作文 | 给定话题自由发挥 | 先亮明观点，再分点论述 |
| 读后续写 | 给定文章开头 | 与原文衔接，注意人称和时态一致 |
| 演讲稿 | 给定演讲场景 | 注意称呼语和结束语，语言有感染力 |

#### 2.2 抓关键信息

- 找出题目中 **必须覆盖的要点**（尤其是中考/高考作文，每个要点都必须写到）
- 确定 **人称**（第一人称 / 第三人称）
- 确定 **时态**（一般现在时 / 一般过去时 / 一般将来时 / 混合时态）
- 确定 **字数要求**

#### 2.3 明确限制

- 检查是否有特定格式要求（如邮件格式、通知格式）
- 是否要求使用特定词汇或句型
- 是否有禁止事项（如"不得透露个人信息"）

---

### 3. 语言与素材准备（内部步骤，不单独输出文件）

#### 3.1 构思立意

确定文章的核心思想和写作角度：

- 明确 **要表达的主旨**
- 确定 **每段的中心句（topic sentence）**
- 规划 **段落结构**（通常三段式：开头引入 → 正文展开 → 结尾总结）

#### 3.2 语言素材准备

为写作准备合适的语言素材：

| 素材类型 | 说明 | 示例 |
|---------|------|------|
| **高分替换词** | 用高级词汇替换平庸词汇 | like → be fond of / be keen on |
| **万能句型** | 适用于多种题目的好句 | It is widely acknowledged that... |
| **连接词** | 让文章逻辑通畅 | however, therefore, in addition |
| **写作模板** | 特定题型的固定框架 | 书信开头：I'm writing to... |
| **名言/谚语** | 为文章增色 | As the saying goes, ... |

#### 3.3 常用高分表达积累

**初中常用高分表达：**

| 普通表达 | 高分替换 |
|---------|---------|
| I think | In my opinion / As far as I'm concerned |
| very good | excellent / outstanding / remarkable |
| like | be fond of / be crazy about / have a passion for |
| many | a large number of / plenty of / a variety of |
| important | vital / essential / significant |
| want to | be eager to / long to / desire to |
| because | owing to / due to / on account of |

**高中常用高分句型：**

| 句型类型 | 示例 |
|---------|------|
| 强调句 | It is ... that ... |
| 倒装句 | Not only ... but also ... / Only by doing ... can we ... |
| 非谓语 | Faced with ..., we should ... / To begin with, ... |
| 定语从句 | ... , which makes me feel ... |
| 虚拟语气 | If I were you, I would ... |
| with 复合结构 | With the development of technology, ... |

### 4. 生成文件一：`作文要点.md`

使用 `write_to_file` 在目标目录下创建此文件。

将整篇作文浓缩为 **5–8 个核心要点**，每个要点用 **1–2 句话** 总结，同时给出对应的英文关键表达。

文件内容格式：

```markdown
# 作文要点：{题目}

> 学段：{初中/高中} | 题型：{书信/通知/话题作文等} | 时态：{主要时态} | 字数：{要求范围}

1. **[要点标题]**：[1-2句话说明这个要点要写什么] | 关键表达：`[对应英文短语/句型]`
2. **[要点标题]**：[1-2句话说明] | 关键表达：`[对应英文短语/句型]`
3. **[要点标题]**：[1-2句话说明] | 关键表达：`[对应英文短语/句型]`
4. **[要点标题]**：[1-2句话说明] | 关键表达：`[对应英文短语/句型]`
5. **[要点标题]**：[1-2句话说明] | 关键表达：`[对应英文短语/句型]`
...

## 加分项提醒

- [列出 2-3 个可以让作文加分的技巧，如使用某种句型、某个高级词汇等]
```

要点撰写规则：

- **第1个要点** 对应开头段，说明如何切入主题（含格式要求，如书信称呼）
- **中间要点** 对应文章主体，每个要点覆盖题目给出的一个写作要求
- **最后一个要点** 对应结尾段，说明如何收束全文
- 每个要点必须附上 **关键英文表达**，帮助学生直接使用
- 要覆盖题目要求的 **所有要点**，一个都不能漏

---

### 5. 生成文件二：`写作思路.md`

使用 `write_to_file` 在目标目录下创建此文件。

详细说明 **如何思考这篇英语作文**，从拿到题目到下笔的完整思维过程。重点教学生"怎么想"和"怎么组织语言"。

文件内容格式：

```markdown
# 写作思路：{题目}

> 学段：{初中/高中} | 推荐题型处理方式：{对应策略}

## 第一步：审题——找出必写内容
[引导学生圈出题目中的关键信息：人称、时态、必须覆盖的要点]

## 第二步：定框架——决定写几段、每段写什么
[帮助学生规划段落结构，明确每段的 topic sentence]

## 第三步：想表达——用英文怎么说
[针对每个要点，给出中文想法到英文表达的转换思路，避免中式英语]

## 第四步：加亮点——让作文脱颖而出
[建议 2-3 个可以使用的高分句型或高级词汇，以及在哪里使用]

## 第五步：检查——写完后看什么
[提供检查清单：时态一致性、主谓一致、拼写、要点覆盖、字数]

## 常见踩坑提醒
[列出同学们容易犯的 3-5 个典型错误，附正误对比]
```

写作思路的撰写规则：

- 用 **对话式语气**，亲切友好，像老师在课堂上一对一辅导
- 重点解决 **中文思维到英文表达的转换**，这是中国学生最大的痛点
- 每个步骤都要有 **具体的中英对照示例**
- **常见踩坑提醒** 必须包含具体的 **正误对比**（❌ 错误写法 → ✅ 正确写法）
- 适当使用表格对比，方便学生查阅

---

### 6. 生成文件三：`完整范文.md`

使用 `write_to_file` 在目标目录下创建此文件。

根据前面的审题和构思，生成一篇完整的英语范文。

文件内容格式：

```markdown
# {范文标题（英文）}

{英语正文，按学段要求控制字数}

---

## 范文中文翻译

{逐段对应的中文翻译，帮助学生理解}

---

## 范文点评

| 维度 | 评价 |
|------|------|
| **内容完整性** | [是否覆盖所有要点] |
| **语言准确性** | [语法、拼写是否正确] |
| **词汇丰富度** | [使用了哪些高分词汇] |
| **句式多样性** | [使用了哪些高分句型] |
| **篇章连贯性** | [衔接是否自然] |
| **预估得分** | XX / XX 分 |

## 亮点词句摘录

| 亮点表达 | 中文含义 | 可迁移场景 |
|---------|---------|-----------|
| `[高分表达1]` | [中文意思] | [其他题目也能用的场景] |
| `[高分表达2]` | [中文意思] | [其他题目也能用的场景] |
| `[高分表达3]` | [中文意思] | [其他题目也能用的场景] |
...
```

范文要求：

#### 6.1 开头技巧（根据题型选择）

| 题型 | 推荐开头方式 | 示例 |
|------|------------|------|
| 书信 | 说明写信目的 | I'm writing to tell you about... |
| 通知 | 直接说明主题 | In order to..., we are going to... |
| 话题作文 | 引出话题 | Nowadays, ... has become a hot topic. |
| 看图写话 | 描述场景 | As is shown in the picture, ... |
| 演讲稿 | 问候 + 话题 | Good morning, everyone! Today I'd like to talk about... |

#### 6.2 正文要求

- **覆盖所有要点**：题目给出的每个要求都必须在正文中体现
- **逻辑清晰**：使用连接词（first, besides, finally 等）串联段落
- **详略得当**：重点要点多写 2-3 句，次要要点 1 句即可
- **适当发挥**：在覆盖要点的基础上适当扩展，但不可跑题
- **语言地道**：避免中式英语，使用自然的英文表达

#### 6.3 结尾技巧

| 题型 | 推荐结尾方式 | 示例 |
|------|------------|------|
| 书信 | 期待回复 | I'm looking forward to your reply. |
| 通知 | 号召参与 | Everyone is welcome! Don't miss it! |
| 话题作文 | 总结观点 | In a word, ... plays an important role in... |
| 演讲稿 | 感谢听众 | Thank you for your listening! |

#### 6.4 语言标准

| 学段 | 词汇要求 | 句型要求 | 语法要求 |
|------|---------|---------|---------|
| **初中** | 课标词汇 + 少量高分替换 | 简单句为主，2-3 个复合句 | 基础语法正确无误 |
| **高中** | 大量高级替换 + 短语运用 | 复合句为主，含非谓语/倒装 | 语法丰富且准确 |

## Customization

- 如果用户指定了题型（如"写一封邮件"），直接按指定题型执行
- 如果用户给出了自己的写作想法或素材，优先围绕用户提供的内容展开
- 如果用户需要针对多个题目进行分析，可以逐一分析，每个题目生成一组三个文件
- 如果用户只需要部分文件（如"只要范文"），则只生成对应文件
- 如果用户要求修改某个文件的内容，只重新生成该文件
- 如果用户要求提供 **多篇范文**（如不同立意/不同题型），可以在 `完整范文.md` 中提供多个版本

## Important Notes

- 三个文件必须保持一致性：要点、思路、范文应基于相同的审题和构思
- **英语表达必须地道准确**，不可出现语法错误或中式英语
- 初中阶段用简单词汇和句型，但确保正确；高中阶段追求多样化和高级感
- `写作思路.md` 用中文书写（因为是教学生怎么思考），但关键表达给出英文
- `完整范文.md` 正文用英文书写，附中文翻译和点评
- `作文要点.md` 用中文说明要点，附英文关键表达
- 范文字数严格控制在学段要求范围内
- **亮点词句摘录** 要注明"可迁移场景"，帮助学生举一反三
- 使用 `write_to_file` 工具创建文件，不要仅在聊天中输出内容
- 常见踩坑提醒必须提供 **正误对比**，只说"注意时态"是不够的
