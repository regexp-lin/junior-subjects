## 一、知识点精讲

不规则动词是中考英语的**必考大户**，很多同学背得头晕眼花却还是踩坑。其实，只要按**四大变位规律**分类记忆，效率翻倍！

### 📌 AAA型（三个一样——原形 = 过去式 = 过去分词）

| 动词 | 过去式 | 过去分词 | 易错提示 |
|------|--------|----------|----------|
| cut | cut | cut | 切割 |
| put | put | put | 放（和 set 易混） |
| cost | cost | cost | 花费 |
| let | let | let | 让 |
| hurt | hurt | hurt | 伤害 |
| set | set | set | 设置、放 |
| shut | shut | shut | 关闭 |
| hit | hit | hit | 击打 |
| **read** | **read** | **read** | ⚠️ 拼写一样，**发音变**！/riːd/→/red/ |

### 错因深度解析

**第1题** ✅ **B**
- **陷阱**：cost 的过去式容易写成 costed，但 cost 是 **AAA型**，三态一致！
- **解析**：bought 是 buy 的过去式（ABB型），cost 的过去式还是 cost。整句：昨天姐姐买了条新裙子，花了200元。

**第2题** ✅ **C**
- **陷阱**：tell 容易误用 telled，实际是 tell-told-told（ABB型）。
- **解析**：last week 表过去，用 told。

**第3题** ✅ **C**
- **陷阱**：do-did-done 是 ABC型，has 后必须接过去分词 done。
- **解析**：has done = 已经做完，完成时结构。

**第4题** ✅ **C**
- **陷阱**：go 的过去分词是 gone，但"去过某地"用 **have been to**，表示"去过又回来了"；have gone to 表示"去了还没回"。
- **解析**：three times 说明去过多次、已经回来，用 has been to。

**第5题** ✅ **B**
- **陷阱**：ride-rode-ridden，昨天用过去式 rode。
- **解析**：yesterday 明确过去，用 rode。

**第6题** ✅ **B**
- **陷阱**：lose 的过去式是 lost，没有 losed！
- **解析**：lose-lost-lost（ABB型），"我丢了钥匙"用 lost。

**第7题** ✅ **C**
- **陷阱**：sit 的过去式是 sat，不是 sitted！
- **解析**：sit-sat-sat（ABB型），小女孩坐在地上哭。

**第8题** ✅ **B**
- **陷阱**：read 是 AAA型，但**发音会变**！过去式读 /red/。
- **解析**：every night 表习惯，用一般现在时 reads；若表"昨晚读了"则用 read（过去式）。

**第9题** ✅ **C**
- **陷阱**：blow-blew-blown，fall-fell-fallen，都是 ABC型。
- **解析**：last night 用过去式，wind blew，trees fell。

**第10题** ✅ **B**
- **陷阱**：begin-began-begun，若表"将在3点开始"可用 will begin。
- **解析**：根据句意"我们在3点开始了会议"为过去，用 began；若理解为将来"我们将在3点开始"则选 D。

**第11题** ✅ **B**
- **陷阱**：leave 的过去式是 left，没有 leaved！
- **解析**：leave-left-left（ABB型），"他把钱包落在公车上了"。

**第12题** ✅ **B**
- **陷阱**：begin-began-begun，see-saw-seen。
- **解析**：两个动作都是昨天发生的，都用过去式：began、saw。

**第13题** ✅ **B**
- **陷阱**：write 的过去式是 wrote，不是 writed！
- **解析**：write-wrote-written（ABC型），last Sunday 用 wrote。

**第14题** ✅ **A**
- **陷阱**：learn 是规则动词，learned/learnt 都可以；speak-spoke-spoken。
- **解析**：for five years 用现在完成时 has learned；now 用一般现在时 speaks。

**第15题** ✅ **A**
- **陷阱**：rise 的过去式是 rose，没有 rised！
- **解析**：rise-rose-risen（ABC型），when we arrived 表过去，太阳已升起用 rose。

**第16题** ✅ **B**
- **陷阱**：forget 的过去式是 forgot，不是 forgetted！
- **解析**：forget-forgot-forgotten（ABC型），yesterday 用 forgot。

**第17题** ✅ **B**
- **陷阱**：build-built-built，cost-cost-cost（AAA型无 costed）。
- **解析**：built 房子，cost 钱，两处都要用正确的不规则形式。

**第18题** ✅ **B**
- **陷阱**：fly-flew-flown，see-saw-seen。
- **解析**：a moment ago 表过去，鸟飞走了用 flew，我看见用 saw。

**第19题** ✅ **B**
- **陷阱**：hear-heard-heard，sound 是规则动词 sounded。
- **解析**：两个动作都是过去的：听见了、听起来像，都用过去式。

**第20题** ✅ **B**
- **陷阱**：swim-swam-swum，feel-felt-felt。
- **解析**：for two hours 可能用完成时 has swum，但"They ___ very tired"需要过去式 felt，表示游完后感到累；前半句若用 swam 表"他们游了两小时"也可。综合考虑，swam 和 felt 都正确。✅
