# ⚖️ 动词形式综合判断——时态/语态/非谓语的三重抉择 (Verb Form Decision Making)

> 日期：2026年08月24日 | 语法分类：综合 | 难度：较难

---

## 一、知识点精讲

### 📌 为什么动词形式是中考难点？

在完形填空、语法填空和短文改错中，**动词形式**是最常考的考点。一个空可能填**谓语动词**（涉及时态、语态），也可能填**非谓语动词**（to do / doing / done）。许多同学看到动词题就发懵，不知道该从何下手。本文将教你一套**三步判断法**，让你有条理地做出正确选择。

---

### 🎯 第一步：判断是谓语还是非谓语

**核心问题**：这个空所在的句子/从句中，**是否已有谓语动词**？

| 情况 | 判断 | 示例 |
|------|------|------|
| 句中无其他谓语 | 此空填**谓语** | He _____ (go) to school. → goes（缺谓语） |
| 句中已有谓语 | 此空填**非谓语** | I saw him _____ (run). → running（saw 是谓语） |
| 有连词引导从句 | 从句需**谓语** | When he _____ (come), I'll tell him. → comes |
| 无连词，有逗号 | 可能是**非谓语** | ____ (hear) the news, she cried. → Hearing |

**技巧**：先找**主语**，再看主语后是否已有**限定动词**（能做谓语的动词形式）。若没有，则此空为谓语；若已有，则此空为非谓语或其他成分。

---

### 🎯 第二步：若是谓语 → 看时态和语态

| 判断顺序 | 考虑因素 | 例句 |
|----------|----------|------|
| 1. 时间状语 | yesterday → 过去时；tomorrow → 将来时；often → 一般现在时 | He _____ (visit) Beijing yesterday. → visited |
| 2. 主从句时态一致 | 主句过去时，从句常过去时或过去完成时 | She said she _____ (be) tired. → was |
| 3. 语态 | 主语是动作承受者 → 被动语态 be + done | The window _____ (break) by the boy. → was broken |
| 4. 主谓一致 | 第三人称单数 + 一般现在时 → 动词加 s | He _____ (like) apples. → likes |

| 时态 | 主动形式 | 被动形式 |
|------|----------|----------|
| 一般现在 | do/does | am/is/are + done |
| 一般过去 | did | was/were + done |
| 一般将来 | will do | will be + done |
| 现在进行 | am/is/are + doing | am/is/are being + done |
| 过去进行 | was/were + doing | was/were being + done |
| 现在完成 | have/has + done | have/has been + done |
| 过去完成 | had + done | had been + done |

---

### 🎯 第三步：若非谓语 → 看 to do / doing / done

| 形式 | 典型用法 | 例句 |
|------|----------|------|
| **to do** | 目的、将来、未发生；want/ask/tell 等后 | I want to go. He asked me to help. |
| **doing** | 主动、进行、已发生；enjoy/finish/mind 等后；作状语表伴随 | I enjoy swimming. He sat there, reading. |
| **done** | 被动、完成；作状语表被动 | I had my hair cut. Seen from the hill, the city is beautiful. |

| 常见搭配 | 后接形式 | 例句 |
|----------|----------|------|
| want, decide, hope, plan | to do | I want to leave. |
| enjoy, finish, mind, avoid | doing | I finished reading the book. |
| see, watch, hear + 宾语 | do（主动全过程）/ doing（主动进行中） | I saw him run / running. |
| see, watch, hear + 宾语 | done（被动） | I saw the window broken. |
| have sth. done | done | I had my bike repaired. |
| be busy, spend time | doing | I'm busy doing homework. |
| 作目的状语 | to do | To pass the exam, he studied hard. |
| 作原因状语（主动） | doing | Being ill, he stayed home. |
| 作原因状语（被动） | done | Scared by the dog, he ran away. |

---

### 📋 综合例题分析

**例 1**：When I _____ (get) home yesterday, my mother _____ (cook) dinner.

- 第一步：两个空都在 when 引导的从句和主句中，各需谓语。
- 第二步：yesterday 表过去；从句"当我到家时"用一般过去时 got；主句"妈妈正在做饭"用过去进行时 was cooking。
- **答案**：got；was cooking

**例 2**：The book _____ (write) by Lu Xun is very popular.

- 第一步：主句主语 The book，谓语是 is，故 _____ (write) 不是谓语。
- 第三步：book 与 write 为被动关系（书被写），用 done 作定语。written
- **答案**：written

**例 3**：He is always the first _____ (arrive) at school.

- 第一步：谓语是 is，the first 后的空为非谓语。
- 第三步：the first to do 固定搭配，表"第一个做某事的人"。
- **答案**：to arrive

---

### ⚠️ 易错点速查表

| 易错点 | 正确思路 | 示例 |
|--------|----------|------|
| 有逗号就填 doing | 看逻辑主语与动词关系 | ____ (see) from space, Earth is blue. → Seen（被动） |
| 看到 to 就填 to do | 可能是介词 to + doing | I'm used to getting up early. |
| 主语与动词被动却用 doing | 被动用 done | The bridge _____ (build) last year. → built |
| 忽略主谓一致 | 三单加 s | She _____ (read) every day. → reads |

### 🔍 错因深度解析

**第1题 (B)**
📌 考点：谓语动词，过去进行时。when 引导的时间状语从句用一般过去时（came），主句表示"当……时正在做某事"，用过去进行时 was watching。
❌ 易错点：选 A watched 表示"看了"已完成，与 when his mother came in（进来那一刻）的"正在看"不符；选 C/D 时态错误。

---

**第2题 (C)**
📌 考点：非谓语动词，see sb. doing 表"看见某人正在做"。句中已有谓语 saw，空处填非谓语。see sb. doing 强调"正在进行的动作"。
❌ 易错点：选 A see sb. do 表"看见某人做了某事"（全过程），此处 in the library 暗示持续状态，doing 更合适；选 B see 后不接 to do；选 D studied 不能作宾语补足语。

---

**第3题 (B)**
📌 考点：谓语动词，被动语态。The room 是"被打扫"的对象，every day 表经常性，用一般现在时被动 is cleaned。
❌ 易错点：选 A cleans 主动语态，room 不能自己打扫；选 C/D 非谓语或形式错误。

---

**第4题 (B)**
📌 考点：非谓语作原因状语，主动用 doing。She 与 hear 为主动关系，Hearing the bad news 表"听到坏消息后"。
❌ 易错点：选 A Heard 表被动，但 she 是"听"的动作发出者；选 C To hear 表目的，与"忍不住哭"的因果关系不符；选 D 不能作状语。

---

**第5题 (C)**
📌 考点：非谓语，enjoy 后接 doing。enjoy doing 为固定搭配。
❌ 易错点：选 A/B enjoy 不接原形或 to do；选 D enjoyed 为谓语形式，enjoy 已有，不能重复。

---

**第6题 (B)**
📌 考点：谓语动词，被动语态 + 一般过去时。The bridge 是"被建造"的对象，in 2020 表过去时间点。
❌ 易错点：选 A built 缺乏 be 动词，不能构成谓语；选 C is built 表经常性，与 in 2020 不符；选 D has built 主动语态错误。

---

**第7题 (B)**
📌 考点：非谓语，too...to 结构。He is too young to go to school = 他年纪太小不能上学。
❌ 易错点：选 A go 不能直接接在 young 后；选 C going 不符合 too...to 结构；选 D goes 为谓语，与 is 冲突。

---

**第8题 (A)**
📌 考点：非谓语作定语，主动用 doing。The man 与 talk 为主动关系，talking to you 表"正在和你说话的人"。
❌ 易错点：选 B talked 表被动或完成，与主动"说话"不符；选 C to talk 表将来或目的，作定语时不如 doing 自然；选 D talk 不能作定语。

---

**第9题 (A)**
📌 考点：have sth. done 结构。have my hair cut = 请别人帮我理发。
❌ 易错点：选 B to cut have 后接省略 to 的不定式作宾补时，是 have sb. do，此处是 have sth. done；选 C cutting 表主动进行；选 D 形式错误。

---

**第10题 (C)**
📌 考点：谓语动词，过去完成时。by the time we arrived 表"到我们到达时"，主句动作在"到达"之前已完成，用 had started。
❌ 易错点：选 A started 与 arrived 同时，无法体现"会议已开始"；选 B has started 与 arrived 过去时不一致；选 D was starting 表"正在开始"，时态不当。

---

**第11题 (B)**
📌 考点：非谓语，ask sb. to do。ask 后接不定式作宾补。
❌ 易错点：选 A help 作宾补时，在 ask 后需 to help；选 C helping ask 不接 doing；选 D helped 为过去式，不能作宾补。

---

**第12题 (A)**
📌 考点：非谓语作状语，被动用 done。the city 与 see 为被动关系（城市被看），Seen from the top 表"从山顶看去"。
❌ 易错点：选 B Seeing 表主动，但"城市"不能主动"看"；选 C To see 表目的；选 D 不能作状语。

---

**第13题 (B)**
📌 考点：谓语动词，被动语态。The window 是"被打破"的对象，用 was broken。
❌ 易错点：选 A broke 主动语态，window 不能自己 break；选 C breaks 时态语态均错；选 D is breaking 表进行，与 has to pay 语境不符。

---

**第14题 (D)**
📌 考点：非谓语，be sorry to have done 表"为已经做过的事抱歉"。让人等了很久是已完成的事。
❌ 易错点：选 A to keep 表将来或一般性；选 B keeping 不作 be sorry 的补足语；选 C kept 形式错误；D to have kept 正确表"已经让……等了"。

---

**第15题 (B)**
📌 考点：非谓语作定语，主动用 doing。The boy 与 sit 为主动关系，sitting under the tree 表"坐在树下的男孩"。
❌ 易错点：选 A sat 为谓语形式，句中已有 is；选 C to sit 表目的或将来；选 D sits 为谓语，与 is 冲突。

---

**第16题 (B)**
📌 考点：谓语动词，现在完成时。since 2010 与现在完成时连用，表"自 2010 年以来一直住在"。
❌ 易错点：选 A lived 过去时不能与 since 连用；选 C lives 一般现在时不能表"持续至今"；选 D is living 进行时与 since 搭配不如 have lived 自然。

---

**第17题 (B)**
📌 考点：非谓语，look forward to 中 to 为介词，后接 doing。look forward to seeing you = 期盼见到你。
❌ 易错点：选 A see 介词 to 后不接原形；选 C/D 形式错误。

---

**第18题 (B)**
📌 考点：非谓语作定语，被动用 done。The story 与 write 为被动关系，written by the writer 表"由作者写的故事"。
❌ 易错点：选 A wrote 表主动；选 C writing 表主动进行；选 D to write 表目的或将来。

---

**第19题 (A)**
📌 考点：make sb. do，使役动词后接省略 to 的不定式。make me clean = 让我打扫。
❌ 易错点：选 B make 后宾补不加 to；选 C cleaning make 不接 doing；选 D cleaned 表被动或完成，不符。

---

**第20题 (C)**
📌 考点：非谓语作状语，完成式表先发生。他先完成作业，再出去玩。Having finished 表"完成"在"出去玩"之前发生。
❌ 易错点：选 A Finished 过去分词表被动或状态，不强调先后；选 B Finishing 表与谓语同时，逻辑不符；选 D To finish 表目的，与 went out 的先后关系不明确。

---

🎯 **今日总结**：做题时**先判谓语还是非谓语**——句中无谓语则填谓语，有则填非谓语。**谓语**看**时态**（时间状语、主从句一致）和**语态**（主动/被动）。**非谓语**看** to do**（目的、将来、固定搭配）、**doing**（主动、进行、enjoy/finish 等后）、**done**（被动、have sth. done、作状语表被动）。先定"谓语/非谓语"，再定形式！💪
