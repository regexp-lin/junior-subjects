# 🔄 each vs every / all vs both / none vs neither 不定代词进阶 (Advanced Indefinite Pronouns)

> 日期：2026年08月15日 | 语法分类：代词 | 难度：中等

---

## 一、知识点精讲

### 📌 什么是不定代词？

不定代词（Indefinite Pronouns）指**不明确指代特定人或物**的代词，如 each, every, all, both, none, neither, either 等。它们常与名词连用，或单独使用，易在**数量范围**和**主谓一致**上出错。

---

### 🎯 each vs every

| 对比 | each | every |
|------|------|-------|
| 强调 | **个体**，逐个 | **整体**，全体 |
| 能否单独用 | ✅ 可单独作主语/宾语 | ❌ 不能单独用，须接名词 |
| 适用数量 | 两个或两个以上 | 三个或三个以上 |
| 谓语 | 单数 | 单数 |
| 典型搭配 | each of + 复数名词 | every + 单数名词 |

| 例句 | 说明 |
|------|------|
| **Each** of the students has a book. | each 可单独用，接 of |
| **Each** student has a book. | each + 单数名词 |
| **Every** student has a book. | every + 单数名词，不能 every of |
| I gave **each** a present. | each 单独作宾语 ✅ |
| I gave **every** a present. | every 不能单独用 ❌ |

**易错**：every 后接单数可数名词，谓语用单数；every 不能接 of 结构（要用 every one of）。

---

### 🎯 all vs both

| 对比 | all | both |
|------|-----|------|
| 范围 | **三个或三个以上**（全部） | **两个**（两者都） |
| 结构 | all + 复数可数/不可数 | both + 复数可数 |
| all of / both of | all of the students | both of the boys |
| 谓语 | 复数（指人/物时） | 复数 |
| 否定 | not all 部分否定 | not both 部分否定 |

| 例句 | 说明 |
|------|------|
| **All** the students are here. | all 指三个及以上 |
| **Both** my parents are teachers. | both 指两个 |
| **All** of them agreed. | all of + 复数 |
| **Both** of us like music. | both of + 复数 |
| **All** the money was lost. | all + 不可数，谓语单数 |

---

### 🎯 none vs neither vs either

| 代词 | 含义 | 范围 | 谓语 | 典型用法 |
|------|------|------|------|----------|
| **none** | 一个都没有 | 三个及以上 | 单复数皆可 | none of + 复数 |
| **neither** | 两者都不 | 两个 | 单数 | neither of + 复数 / neither + 单数 |
| **either** | 两者中任一 | 两个 | 单数 | either of / either...or |

| 例句 | 说明 |
|------|------|
| **None** of the books is/are interesting. | none 三者及以上，谓语可用单或复 |
| **Neither** of the two answers is correct. | neither 两者都不 |
| **Either** of the two will do. | either 两者中任一个 |
| **Neither** student passed. | neither + 单数名词 |

---

### 🎯 部分否定 vs 全部否定

| 类型 | 结构 | 含义 | 例句 |
|------|------|------|------|
| **部分否定** | not all | 并非全部 | Not all birds can fly. 并非所有鸟都会飞。 |
| **部分否定** | not both | 并非两者都 | Not both of them are right. 并非两个都对。 |
| **部分否定** | not every | 并非每个 | Not every student likes maths. 并非每个学生都喜欢数学。 |
| **全部否定** | none | 一个都没有 | None of them can fly. 他们没人会飞。 |
| **全部否定** | neither | 两者都不 | Neither of them is right. 他们俩都不对。 |

**注意**：not all = 有些……不；none = 全部不。区分清楚！

---

### 📋 主谓一致速查

| 代词 | 谓语形式 |
|------|----------|
| each / every + 单数名词 | 单数 |
| each of / neither of / either of + 复数 | 单数（正式） |
| all / both + 复数名词 | 复数 |
| none of + 复数 | 单数或复数（口语常用复数） |
| all + 不可数名词 | 单数 |

---

### ⚡ 易错点汇总

| 易错点 | 正确 | 错误 |
|--------|------|------|
| every 单独用 | Every student came. | Every came. ✗ |
| each of 谓语 | Each of them is... | Each of them are... （正式用单数） |
| both 指两个 | Both (of) the two) | Both the three ✗ |
| not all 部分否定 | Not all are good. | All are not good. （歧义） |
| neither 谓语 | Neither is right. | Neither are right. ✗ |

### 🔍 错因深度解析

**第1题 (D)**
📌 考点：the two girls 指两个，neither 表"两者都不"，与单数谓语 is 连用。
❌ A each 强调个体，each of the two 也可但 neither 更贴合"两个都不"；B every 不接 of the two；C both 谓语用复数 are。

---

**第2题 (A/B)**
📌 考点：each/every + 单数名词，谓语用单数。Each student 和 Every student 均可。
❌ C all 后接复数；D both 指两个。

---

**第3题 (B)**
📌 考点：the answers 若理解为两个，neither 表"两个都不对"。
❌ A none 指三个及以上；C either 表"任一"；D both 表"两者都"，与 is correct 矛盾。

---

**第4题 (A)**
📌 考点：Not all people 部分否定，表"并非所有人都喜欢"。
❌ B not both 用于两者；C not every 后接单数名词（Not every person）；D not each 较少用。

---

**第5题 (A)**
📌 考点：Each of the boys 表"每个男孩"，谓语用单数 has。
❌ B every 不能接 of the boys（要用 every one of）；C all 谓语用复数；D both 谓语用复数。

---

**第6题 (C)**
📌 考点：two brothers，both 表"两个都是"，谓语用复数 are。
❌ A each 谓语用单数；B every 不能单独作主语；D neither 表"两者都不"。

---

**第7题 (A)**
📌 考点：the three books 指三个，none 表"一本都没有趣"。
❌ B neither 只用于两者；C either 表"任一"；D both 只用于两者。

---

**第8题 (A)**
📌 考点：each side 表"每一边"（街道有两边），each 强调个体。
❌ B every side 也可，但 each 更强调"各自"；C all 后接复数；D both sides 也可，但 both 后接复数。

---

**第9题 (C)**
📌 考点：Which one（两个中选一），Either will do 表"任一个都可以"。
❌ A none 表都不要；B neither 表两个都不行；D both 表两个都要，不符语境。

---

**第10题 (D)**
📌 考点：parents 指父母两人，both 表"两人都是"。
❌ A each 谓语用单数；B every 不接复数名词；C all 用于三者及以上。

---

**第11题 (C)**
📌 考点：Each of the students 表"每个学生"，谓语用单数 has。
❌ A none 表"没人通过"；B every 不接 of；D both 指两个。

---

**第12题 (A)**
📌 考点：Not all the students 部分否定，表"并非所有学生都出席了"。
❌ B not both 用于两者；C/D not every/each 后接单数名词。

---

**第13题 (B)**
📌 考点：the two teams，neither 表"两队都没赢"（平局或都输）。
❌ A none 用于三者及以上；C either 表"任一赢"；D both 表"两队都赢"矛盾。

---

**第14题 (A/B)**
📌 考点：each/every child 表"每个孩子"，均可。
❌ C all 后接复数 children；D both 指两个。

---

**第15题 (A)**
📌 考点：None of us 表"我们中没人想去"，谓语用单数 wants。
❌ B neither 指两者；C either 表"任一想"；D both 谓语用复数。

---

**第16题 (A)**
📌 考点：money 不可数，All the money 表"所有的钱"，谓语用单数 was。
❌ B both 接可数复数；C/D each/every 接可数单数。

---

**第17题 (A)**
📌 考点：the sisters 若指两个，Each of the sisters 表"两姐妹各自都结婚了"。
❌ B every 不接 of the sisters；C all 谓语用复数；D neither 表"两个都没结婚"。

---

**第18题 (B)**
📌 考点：Not every student 部分否定，表"并非每个学生都及格"。
❌ A not each 较少见；C not all 后接复数 students；D none 为全部否定。

---

**第19题 (C)**
📌 考点：don't like + either = 两个都不喜欢。either 在否定句中表"任一都不"。
❌ A none 不用于两者；B I don't like neither 双重否定错误；D don't like both 表"并非两个都喜欢"。

---

**第20题 (A)**
📌 考点：All of them 表"他们全部"，谓语用复数 agreed。
❌ B both 指两个；C each 谓语用单数；D every 不能单独作主语。

---

🎯 **今日总结**：不定代词三板斧——**each 可单独 every 不行**（each 重个体 every 重整体）；**all 三个 both 两个**（all/both 配复数）；**none 全无 neither 两不**（none/neither 全部否定）。部分否定用 **not all / not every**，别与 none/neither 混淆！💪
