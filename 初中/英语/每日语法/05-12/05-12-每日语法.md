# 时间的疤痕与习惯的演变——used to 的前世今生与三大分身（used to phrases）

> 日期：2026 年 05 月 12 日 | 语法分类：固定搭配与非谓语动词 | 难度：中等 + 易混淆词组辨析

---

## 一、知识点精讲

`used to`，是初中英语里最容易让人精神分裂的一组词。
它之所以可怕，是因为它不仅有三个长得几乎一模一样的连体婴兄弟，而且这三兄弟每个人的“脾气(后面接的词的词性)”以及“性格(中文翻译)”完全不一样！
在单项选择和完形填空中，只要看见 `used to`，必须启动三级防空警报，做三步走：找前面有没有 be、翻译句子意思、找后面接的是 doing 还是 do！

### 四大门派彻底起底

### 1. 过去修仙者：used to do sth (过去常常做某事)
- **长相特征**：前面干干净净，**没有任何 be 动词或者 get**。
- **中文含义**：表示【过去】经常做某件事，或者过去是一种什么状态，但暗含一个致命潜台词：**现在已经不做了 / 状态变了**！
- **后面接谁**：动词原形 (`do`)。
  - *例：I **used to smoke**, but now I don't. (我过去经常抽烟，但现在不抽了。)*
  - *变式考法(否定)：I didn't use to ... (注意：变否定时借助did，所以used必须还原为use)*
  - *变式考法(疑问)：Did you use to...?*

### 2. 习惯适应者：be used to doing sth / be used to sth (习惯于做某事)
- **长相特征**：前面**必须有 be 动词** (am/is/are/was...) 或者 **get/become**。
- **中文含义**：表示【长期的适应】，意思是“习惯于...”或者“适应了...”。
- **后面接谁**：如果接动词，**必须用 -ing 形式 (`doing`)**！这是最阴险的一招！这里的 to 是【介词】，不是不定式！或者直接接名词和代词。
  - *例接doing：My grandpa **is used to getting up** early. (我爷爷习惯于早起。)*
  - *例接名词：I **am used to** the weather here. (我已经习惯这里的天气了。)*

### 3. 被动工具人：be used to do sth (被用来做某事)
- **长相特征**：前面**也有 be 动词**！这和第2条长得一模一样！！
- **中文含义**：这是个无情的被动语态机器。`use` 恢复它最原始的本义“使用”。翻译为：“主语 (通常是死物) **被用来** 做某事”。
- **后面接谁**：动词原形 (`do`)！因为这里的 to 恢复成了不定式符号 (表示目的)。
  - *例：Wood **is used to make** paper. (木头被用来造纸。主语木头是死物，是被人类用来造纸的！)*

### 4. 衍生材质使者：be used for doing sth (被用来做某事)
- **长相特征**：不仅有 be 动词，连 to 都没了，换成了 **for**。
- **中文含义**：和第3条一模一样，表示事物的用途“被用来做...”。
- **后面接谁**：既然跟着介词 for，动词必须强行变成 **-ing 形式 (`doing`)**！
  - *例：Knives **are used for cutting** things. (刀被用来切东西。)*

### 逐题解析

**第 1 题：答案 A**

- **解析**：句意"我爸爸**过去经常**抽烟，但他现在已经戒了"。
  描述发生在过去的常态或习惯（且现在已经停止）。使用基础词组 **used to do sth**。
  空格后面是原形动词 `smoke`。选项直接匹配 `used to`。选 A。(不能选有be动词的，那是习惯于或者被用来)。
- **考查知识点**：used to do sth 的基本概念及语境。

**第 2 题：答案 B**

- **解析**：句意"那个老人**习惯于**每天晚饭后散步"。
  前面有 `is`！所以构成词组 `be used to (习惯于...)`。
  既然是习惯于，这里的 to 那就是万恶的【介词】！介词后面强吃 **-ing 形式**！
  所以走步 walk 必须加上 ing ——> `walking`。选 B。
- **考查知识点**：be used to doing 强迫动名词的陷阱。

**第 3 题：答案 A**

- **解析**：句意"在世界各地木头**被用来**造纸"。"
  主语木头。它不会自己习惯被造纸！这是被动语态机器：`be used to (被用来...)`。
  当翻译为"被用来"时，这个 to 是表目的的不定式，它是个极其老实的符号！所以后面必须跟最正派的【动词原形】！
  造纸 -> `to make`。选 A。
- **考查知识点**：be used to do 被动语态的不定式接原形。

**第 4 题：答案 B**

- **解析**：句意"你**过去常常**踢足球吗？—是的，但现在我打篮球了"。
  这题考的是 `used to` 的疑问句换装！
  既然借用了助动词 `Did`。那么 used 身上的过去式尾巴 -d 就必须被扒掉，卸妆还原本源：变成 **use to**！选 B。(极其容易眼瞎选A)。
- **考查知识点**：used to 的疑问句还原 use to。

**第 5 题：答案 C**

- **解析**：句意"简前两个月在中国很艰难，但现在她已经**习惯于吃**中国菜了"。
  前面不仅有主语 she 的系动词 `is`，还是一段融入中国的感情戏。这必须是：`be used to doing (习惯于)`。
  吃必须变成吃瓜群众：`eating`。选 C。
- **考查知识点**：be used to doing 第二次出没及感情色彩辨析。

**第 6 题：答案 A**

- **解析**：句意"钢笔**被用于**写字"。"
  如果是 to，后面应该是 write (be used to write)。
  但空格后面的词是 `writing (-ing形式)`！
  能毫无廉耻地强吃 -ing 形式的被动武器，唯有它的死对头胞弟：**be used for doing (被用来做某事)**。填入 for。选 A。
- **考查知识点**：be used for doing (被用来做某事)。

**第 7 题：答案 C**

- **解析**：句意"我**过去常常**晚起，但现在我**已经习惯了**早起"。
  前半句：过去常常（且现在变了）—— `used to` + get (原形)。
  后半句：现在养成习惯了 —— `am used to` (习惯于) + getting (前有be，后吃ing)。
  所以完美串烧组合是：C。
- **考查知识点**：纯历史习惯(used to do)与目前适应(be used to doing)的双轨结合。

**第 8 题：答案 C**

- **解析**：后置定语非谓语的高级伪装。
  句意"那把_____切面包的刀非常锋利"。
  这句话的完整结构是：[ The knife (主语) ] is (系动词) very sharp。
  所以刀后面填的东西，是修饰刀的一个修饰成分："被用来切面包的（刀）"。
  在英语里，被动语态当形容词修饰定语时，不需要写 be 动词！直接省略 as/is，保留过去分词即可（which is used for -> used for）。
  由于后面是 `cutting (做状语必须用for doing接引)`，所以是 **used for** cutting bread。选 C。(不是一个完整的谓语结构，所以不能带is)。
- **考查知识点**：过去分词短语作后置定语的省略（used for doing sth）。

**第 9 题：答案 A**

- **解析**：句意"我们**过去常常**住在一个小山村，但现在我们住在大城市"。"
  标准的怀旧历史感叹句。"曾经沧海...但现在..."。
  原味素颜词组：**used to** + 动词原形 live。选 A。
- **考查知识点**：used to do 描述过去状态与现在的跳水断层。

**第 10 题：答案 B**

- **解析**：句意"她（过去）**不习惯于**住在小镇，所以她搬回了伦敦"。
  主干 `wasn't used to`，既有 be动词的过去式(was)，又有否定。翻译为：“过去一直没能习惯于...”。
  它是 `be used to` 的过去时态魔改！
  既然核心词组是 (习惯于)，那么 to 依然是吃人的介词！吃下 -ing！`living`。选 B。
- **考查知识点**：be used to 拓展过去时被包浆后的强吃 -ing 原理不变。

**第 11 题：答案 C**

- **解析**：句意"在过去，人们**常常**靠邮寄信件，但现在他们**习惯于发**电子邮件了"。
  第一把：过去的一段光阴... `used to + do (send)`。
  第二把：但现在科技发达了，大家都习惯了新玩意儿... `are used to + doing (sending)`。
  组合匹配：C项 (used to; sending)。
- **考查知识点**：双剑合璧再次强化(used to do vs be used to doing)。

**第 12 题：答案 B**

- **解析**：句意"我奶奶过去**从没**自己一个人住过，所以她现在**不怎么习惯**这种安静的生活（等等，不对！so she `is used to` 也就是她现在已经习惯了）"。
  前面想表达的是：过去从没经常... ——> `didn't use to do`。
  所以第一空不仅考察你知不知道是否定过去行为，还考察有没有拔掉 -d 尾巴 (didn't use to)。
  后面表达：现在已经适应了... ——> `is used to`。
  正确答案为 B。(如果A项，助动词不能用 doesn't，因为这说的是过去的事实！)。
- **考查知识点**：过去行为习惯的否定变体 didn't use to do。

**第 13 题：答案 B**

- **解析**：句意"煤**被用来**在很多工厂发电"。
  煤(Coal)是不可数死物。只有两个下场：is used to do (发) 或者 is used for doing (发)。
  看后面那个张牙舞爪带着 ing 的词块：`producing`！！
  那一定是选带领 for 系分支大军的：**is used for**。选 B。
- **考查知识点**：be used for doing 物体性能防守战。

**第 14 题：答案 B**

- **解析**：句意"我找不到那把**被用来**开门的钥匙了"。
  主句为 "I can't find the key(钥匙)"。
  后面修饰那把什么样的钥匙(定语)：被用来开门的。
  被用来：使用过去分词短语作后置定语（省略 which is）。
  因为接着开门动词是 `open`(极其原生的动词原形)。不用 for！！！用 **to**！
  所以就是：`used to`(open the door)。选 B。
- **考查知识点**：被用于(used to do) 省略be后的短语作定语。

**第 15 题：答案 D**

- **解析**：句意"玛丽在中国待了5年了，所以她**习惯于**在这儿的生活"。
  前文定调：待了五年长年累月熏陶。后引伸出一种：深入骨髓的"习惯于"。
  而且紧跟后面的动词是 `living` (-ing形式)。且主语为人。
  只有天字一号词组：**is used to** 符合。(如果选gets used to也可以当习惯于讲，但B项主语是实义，中考倾向性用be，且此题主考状态。最直接的就是 D)。选 D。
- **考查知识点**：以人为主语的习惯融合表达 be used to doing。

**第 16 题：答案 C**

- **解析**：句意"我们学校的图书馆**过去常常是/曾经**非常小，但现在很大了"。"
  描述过去存在的一种客体状态，现在发生了翻天覆地的截距。
  `used to be` (过去曾经是...) 是一个被盘包浆的极其丝滑好用的短语。
  此处的 be 是系动词的原形！在 used to 的加持下当大梁接客！
  所以选用纯净的 **used to**。选 C。
- **考查知识点**：used to 接 系动词 be 表达曾经的状态全貌。

**第 17 题：答案 B**

- **解析**：句意"水能**被用来**洗衣服和清洁地板"。
  主语水 (死物)。被动语态机器无疑！
  前面有一个强悍无比的情态动词 **can (能/可以)**！
  情态动词后面必须接动词祖宗(原形)。所以 be 动词被打出绝对原形 `be`！
  后面接了两个极干净的光头动词动作 wash / clean (原形)。所以这是 `to do` 的疆场。
  接轨下来就是：can **be used to** (wash...)。选 B。
- **考查知识点**：情态动词can + 物品被用来(be used to do)的被动复合结构。

**第 18 题：答案 C**

- **解析**：句意"经过漫长的旅程，他们终于**习惯了在**帐篷里睡觉"。"
  这里的主将是 `got used to`。它是 `be used to` 的兄弟，意思是：从不习惯，渐渐变得（习惯起来了）。
  但它的核心法制跟大哥一模一样！那个 to，不管前面怎么改朝换代，**依然是一个饿死鬼介词！**
  所以后方的动作必须全部变成 -ing 放血献祭！`sleeping` ！
  组合出来：**to sleeping**！选 C。(绝命深坑题！无数人没选带to的，或者没选带ing的被双杀！)。
- **考查知识点**：get used to 后接介词to的组合及动名词祭品(sleeping)。

**第 19 题：答案 B**

- **解析**：句意"当你在北京的时候，你**以前是/曾经是**个老师吗？"
  这题是将 16题的 (used to be) 强行拔掉毛发塞进了疑问句照妖镜里！
  疑问句照妖镜 `Did ... ?`  它会扒掉用于过去特征的各种后缀！
  所以 used to be 会被照出原形：**use to be** (曾经做过/曾经是...)。选 B。
- **考查知识点**：used to be 作为系表过去的疑问句全形态还魂。

**第 20 题：答案 C**

- **解析**：综合辨析大演练最后一战！
  句意"邮票在**过去被用来**寄信件，但现在几乎没人用了。
  1. 找时态：in the past (在过去)，所以用 past tense 的 be 动词。
  2. 找性质：邮票 (死物)，被用来的被动发作。主语复数。 -> **were used**。
  3. 找对接端口：后面是极度张牙舞爪挥舞进行时的 `sending`！
  被用来做 + -ing：必然选用介词宗门的分支头领 **for**！
  全部拼接：**were used for**。选 C。
- **考查知识点**：过去事理的被动(were used) 与 for doing 联合演练阵仗。
