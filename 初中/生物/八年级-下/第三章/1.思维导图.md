# 八年级下册生物 第三章 思维导图（树状图）

- 第三章（按 prompt 对应 `ba-xia-new_55-80.pdf`）
  - 生物的进化
    - 一、进化证据
      - 1. 化石证据
        - 地层越古老，化石生物结构越简单
        - 地层越新，生物类型越复杂
      - 2. 解剖学证据
        - 同源器官说明共同祖先
      - 3. 胚胎学与分子生物学证据
        - 早期胚胎相似性
        - DNA/蛋白质相似性
    - 二、进化历程
      - 生物由简单到复杂
      - 由低等到高等
      - 由水生到陆生
    - 三、自然选择学说
      - 1. 过度繁殖
      - 2. 生存斗争
      - 3. 遗传和变异
      - 4. 适者生存，不适者被淘汰
    - 四、人工选择
      - 人类按需求选择育种
      - 可在较短时间内改变品种性状
    - 五、生物进化与环境
      - 环境变化推动进化方向
      - 生物与环境协同演化
