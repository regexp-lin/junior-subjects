# 八年级下册生物 第一章 思维导图（树状图）

- 第一章（按 prompt 对应 `ba-xia-new_6-23.pdf`）
  - 生物的生殖和发育
    - 一、植物的生殖
      - 1. 有性生殖
        - 经过两性生殖细胞结合形成受精卵
        - 后代具有双亲遗传特征
      - 2. 无性生殖
        - 不经过两性生殖细胞结合
        - 方式：扦插、嫁接、压条、组织培养等
        - 保持亲本优良性状
    - 二、昆虫的生殖和发育
      - 1. 生殖方式：有性生殖，体内受精
      - 2. 发育类型
        - 完全变态：卵→幼虫→蛹→成虫（如家蚕、蝴蝶）
        - 不完全变态：卵→若虫→成虫（如蝗虫）
    - 三、两栖动物的生殖和发育
      - 多在水中受精和发育
      - 发育过程：受精卵→蝌蚪→幼蛙→成蛙
      - 变态发育与水环境关系密切
    - 四、鸟类的生殖和发育
      - 求偶、交配、筑巢、产卵、孵卵、育雏
      - 鸟卵结构适应陆地发育
      - 胚盘是胚胎发育部位
