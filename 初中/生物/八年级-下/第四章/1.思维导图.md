# 八年级下册生物 第四章 思维导图（树状图）

- 第四章（按 prompt 对应 `ba-xia-new_81-99.pdf`）
  - 传染病和免疫
    - 一、传染病
      - 1. 概念
        - 由病原体（细菌、病毒、真菌、寄生虫等）引起
        - 可在人与人或人与动物之间传播
      - 2. 三个基本环节
        - 传染源
        - 传播途径
        - 易感人群
      - 3. 预防措施（对应三环节）
        - 控制传染源
        - 切断传播途径
        - 保护易感人群
      - 4. 常见类型
        - 呼吸道传染病
        - 消化道传染病
        - 血液传染病
        - 体表传染病
    - 二、人体的免疫
      - 1. 非特异性免疫（先天）
        - 第一防线：皮肤和黏膜
        - 第二防线：体液杀菌、吞噬细胞吞噬
      - 2. 特异性免疫（后天）
        - 抗原刺激后产生
        - 淋巴细胞参与
        - 抗体与抗原特异性结合
      - 3. 免疫器官与细胞
        - 胸腺、脾、淋巴结
        - 淋巴细胞、吞噬细胞
      - 4. 计划免疫
        - 通过接种疫苗获得免疫力
        - 疫苗本质：减毒或灭活病原体及其成分
    - 三、免疫的利与弊
      - 1. 有利：防御病原体、清除体内异常细胞
      - 2. 不利：免疫过强可导致过敏、器官移植排斥等
    - 四、健康生活与防病
      - 良好卫生习惯
      - 佩戴口罩、勤洗手、通风
      - 科学接种疫苗
      - 早发现、早报告、早隔离、早治疗
