# 七年级上册生物 第二章 详细知识点

> 说明：本章内容对应课本中的“第三章《从细胞到生物体》”。

## 一、第一节：细胞通过分裂产生新细胞

### 1. 细胞生长
1. 细胞通过吸收外界营养物质，并把这些物质转化为组成自身的物质，从而体积增大。
2. 细胞生长有一定限度，不能无限长大。
3. 生物体长大主要依靠：
   - 细胞体积增大（生长）
   - 细胞数目增多（分裂）

### 2. 细胞分裂的基本过程
1. 细胞分裂是一个细胞分成两个细胞的过程。
2. 一般顺序：
   - 细胞核先分裂为两个
   - 细胞质分成两份
   - 在原细胞中央形成新的分隔结构
3. 动物细胞与植物细胞分裂差异：
   - 动物细胞：细胞膜从中部向内凹陷，最终一分为二。
   - 植物细胞：中部先形成新的细胞膜，再形成新的细胞壁。

### 3. 染色体与遗传物质
1. 染色体是细胞核中易被碱性染料染成深色的结构。
2. 染色体主要由 DNA 和蛋白质组成。
3. DNA 是遗传物质，携带遗传信息。
4. 细胞分裂中的关键变化：
   - 分裂前：染色体复制加倍。
   - 分裂时：复制后的染色体平均分配到两个新细胞中。
   - 结果：两个新细胞的染色体形态和数目与原细胞一致。
5. 意义：保证亲代细胞和子代细胞遗传信息相对稳定。

### 4. 本节常考点
1. 细胞数目增加的根本方式是细胞分裂。
2. 染色体变化规律：先复制后平均分配。
3. 判断题易错点：
   - “细胞能无限长大”是错误说法。
   - “分裂后新细胞染色体数目减半”通常不符合本章所述体细胞分裂情形。

## 二、第二节：动物体的结构层次

### 1. 细胞分化
1. 受精卵通过分裂形成大量细胞。
2. 这些细胞在发育过程中会在形态、结构和功能上发生差异，形成不同类型细胞。
3. 这种变化过程叫细胞分化。

### 2. 组织
1. 组织是由形态相似、结构和功能相同的细胞联合在一起形成的。
2. 动物体基本组织：
   - 上皮组织：覆盖体表和器官内表面，具有保护、分泌等功能。
   - 结缔组织：分布广，具有连接、支持、保护、营养等功能（如血液、骨组织等）。
   - 肌肉组织：由肌细胞构成，能收缩和舒张，产生运动。
   - 神经组织：由神经细胞构成，能感受刺激并传导神经冲动。

### 3. 器官与系统
1. 不同组织按照一定次序结合形成器官。
2. 多个器官按一定功能关系构成系统（如消化系统、呼吸系统等）。
3. 各系统在神经和体液调节下协调配合，维持个体生命活动。

### 4. 动物体结构层次总式
1. 细胞 → 组织 → 器官 → 系统 → 动物体。
2. 这是动物体结构层次的标准表述，考试中常要求按顺序填写。

## 三、第三节：植物体的结构层次

### 1. 植物主要组织
1. 分生组织：细胞小、壁薄、核大、分裂能力强，使植物不断生长。
2. 保护组织：分布在器官表面，起保护作用。
3. 营养组织：储藏营养物质，或进行光合作用。
4. 输导组织：运输水、无机盐和有机物。

### 2. 植物器官
1. 营养器官：根、茎、叶。
2. 生殖器官：花、果实、种子。
3. 绿色开花植物通常由六大器官组成：根、茎、叶、花、果实、种子。

### 3. 植物体结构层次总式
1. 细胞 → 组织 → 器官 → 植物体。
2. 与动物相比，植物体一般不具有“系统”这一层次。

### 4. 对比记忆
1. 动物：到“系统”为止再到个体。
2. 植物：组织后直接到“器官”，再到植物体。

## 四、第四节：单细胞生物

### 1. 概念
1. 单细胞生物是指身体只由一个细胞构成的生物。
2. 常见单细胞生物包括草履虫、变形虫、眼虫、部分藻类和酵母菌等。

### 2. 单细胞生物的生命活动
1. 虽然只有一个细胞，但能独立完成营养、呼吸、排泄、运动、生殖等生命活动。
2. 体现了“细胞是生物体结构和功能基本单位”。

### 3. 草履虫结构与功能对应
1. 表膜：进行气体交换。
2. 纤毛：摆动推动身体运动。
3. 口沟：摄食入口。
4. 食物泡：完成食物消化吸收。
5. 收集管和伸缩泡：调节体内水分并排出多余水分。
6. 胞肛：排出消化后的残渣。

### 4. 与人类的关系
1. 有利方面：
   - 参与生态系统物质循环。
   - 可用于科学研究与水质监测。
2. 不利方面：
   - 少数种类可能引发疾病或造成危害。

## 五、本章综合与易错点

### 1. 综合主线
1. 细胞分裂使细胞数量增加。
2. 细胞分化形成组织。
3. 组织进一步构成器官与个体，形成生物体结构层次。

### 2. 高频易错点
1. 混淆“细胞生长”和“细胞分裂”。
2. 遗漏“染色体先复制再平均分配”。
3. 把植物体层次误写成“细胞→组织→器官→系统→植物体”。
4. 认为单细胞生物“结构简单就不进行完整生命活动”。

### 3. 复习建议
1. 用“流程图”记忆细胞分裂过程。
2. 用“对照表”区分动物体与植物体结构层次。
3. 结合草履虫示意图进行结构-功能一一对应记忆。
