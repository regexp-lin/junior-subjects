# 七年级上册生物 第二章 思维导图（树状图）

- 第二章（对应课本：第三章《从细胞到生物体》）
  - 第一节 细胞通过分裂产生新细胞
    - 细胞生长
      - 细胞吸收营养物质
      - 体积由小变大
      - 细胞不能无限长大
    - 细胞分裂
      - 一个细胞分成两个细胞
      - 细胞核先分裂，细胞质再分开
      - 动物细胞：细胞膜向内凹陷缢裂
      - 植物细胞：中部形成新的细胞膜和细胞壁
    - 染色体变化
      - 染色体由 DNA 和蛋白质组成
      - 分裂前染色体先复制
      - 分裂时染色体平均分配到两个新细胞
      - 新细胞与原细胞染色体数目一致
  - 第二节 动物体的结构层次
    - 细胞分化
      - 分裂后细胞在形态、结构、功能上出现差异
      - 形成不同组织
    - 动物四大基本组织
      - 上皮组织：保护、分泌
      - 结缔组织：支持、连接、营养、保护
      - 肌肉组织：收缩和舒张，产生运动
      - 神经组织：接受刺激、传导神经冲动
    - 器官和系统
      - 不同组织按一定次序组合成器官
      - 多个器官协同构成系统
      - 系统协调配合完成生命活动
    - 动物体结构层次
      - 细胞 → 组织 → 器官 → 系统 → 动物体
  - 第三节 植物体的结构层次
    - 植物组织
      - 分生组织：细胞分裂能力强
      - 保护组织：保护内部结构
      - 营养组织：储藏营养物质
      - 输导组织：运输水、无机盐和有机物
    - 植物器官
      - 营养器官：根、茎、叶
      - 生殖器官：花、果实、种子
    - 植物体结构层次
      - 细胞 → 组织 → 器官 → 植物体
      - 植物体一般无“系统”层次
  - 第四节 单细胞生物
    - 常见类型
      - 草履虫、变形虫、眼虫、酵母菌等
    - 主要特点
      - 个体只由一个细胞构成
      - 一个细胞完成营养、呼吸、排泄、运动、生殖等生命活动
    - 草履虫结构与功能
      - 纤毛：运动
      - 口沟：摄食通道
      - 食物泡：消化吸收
      - 表膜：气体交换
      - 收集管和伸缩泡：排出多余水分
      - 胞肛：排出食物残渣
    - 与人类关系
      - 有益：参与物质循环、可作研究材料
      - 有害：部分单细胞生物可致病
