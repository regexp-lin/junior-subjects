# 七年级上册生物 第五章 详细知识点

> 说明：本章内容对应课本中的“第三章《微生物》”。

## 一、第一节：微生物的分布

### 1. 微生物概念与特点
1. 微生物是个体微小、结构简单的一类生物，通常需要借助显微镜观察。
2. 主要特点：
   - 个体微小
   - 数量庞大
   - 繁殖迅速
   - 分布广泛

### 2. 微生物的分布
1. 空气中：尘埃和飞沫可携带微生物。
2. 水体中：河流、湖泊、海洋及生活用水中均可能存在。
3. 土壤中：有机质丰富，微生物种类多。
4. 生物体内外：人体皮肤、口腔、肠道和动植物体表或体内均有微生物。
5. 结论：微生物几乎无处不在。

### 3. 菌落与培养
1. 菌落
   - 指一个或多个细菌或真菌在适宜固体培养基上繁殖形成的肉眼可见群体。
2. 细菌菌落与真菌菌落常见区别
   - 细菌菌落：较小、表面较光滑黏稠、颜色较单一。
   - 真菌菌落：常较大、絮状或绒毛状，颜色多样。
3. 培养细菌/真菌的一般步骤
   - 配制培养基（固体培养基常需加琼脂）
   - 高温灭菌
   - 冷却后接种
   - 置于适宜温度下培养并观察

### 4. 无菌操作与对照实验
1. 无菌操作目的：防止杂菌污染，保证实验结果可靠。
2. 常见操作：器具消毒、接种过程尽量短、及时封口。
3. 对照实验：设置空白对照，用于判断菌落是否来自接种样本。

## 二、第二节：细菌

### 1. 细菌形态与结构
1. 细菌是单细胞生物。
2. 基本结构包括：细胞壁、细胞膜、细胞质和遗传物质集中区域。
3. 重要特征：没有成形的细胞核。
4. 部分细菌有：
   - 荚膜（保护作用）
   - 鞭毛（运动作用）
   - 芽孢（抗逆结构，不是生殖细胞）

### 2. 细菌营养方式
1. 细菌多不能进行光合作用，主要为异养。
2. 异养方式可分为：
   - 腐生：分解动植物遗体或有机物残体。
   - 寄生：从活体生物获取营养。

### 3. 细菌生殖
1. 主要方式是分裂生殖。
2. 条件适宜时繁殖速度很快。
3. 环境恶劣时，部分细菌形成芽孢，提高生存能力。

### 4. 细菌与人类关系
1. 有益作用
   - 乳酸菌用于酸奶、泡菜等发酵。
   - 分解者作用促进物质循环。
   - 污水处理、土壤改良、生物技术中有重要应用。
2. 有害作用
   - 一些致病菌可引起肺炎、肠道感染等疾病。
   - 导致食物腐败变质。

## 三、第三节：真菌

### 1. 真菌主要类型
1. 单细胞真菌：酵母菌。
2. 多细胞真菌：霉菌、蘑菇、木耳等。

### 2. 真菌结构特点
1. 真菌细胞具有细胞核，属于真核生物。
2. 多细胞真菌一般由菌丝构成，菌丝交织形成菌体。
3. 真菌不含叶绿体，不能进行光合作用，多为异养。

### 3. 真菌生殖与生活条件
1. 真菌多通过孢子繁殖。
2. 生长繁殖需要适宜条件：
   - 水分
   - 适宜温度
   - 有机营养
   - 适当环境（部分需氧）

### 4. 真菌与人类关系
1. 有益方面
   - 酵母菌用于馒头、面包、酿酒等发酵生产。
   - 食用菌是重要食物来源。
   - 某些霉菌可生产抗生素（如青霉素相关来源）。
2. 有害方面
   - 食品霉变。
   - 部分真菌可引起皮肤病或其他疾病。

## 四、第四节：病毒

### 1. 病毒的基本特征
1. 病毒无细胞结构。
2. 组成：蛋白质外壳 + 内部遗传物质（DNA 或 RNA）。
3. 个体极其微小，需借助电子显微镜观察。

### 2. 病毒的生活方式
1. 只能寄生在活细胞内。
2. 进入宿主细胞后利用宿主物质和结构完成复制增殖。
3. 离开活细胞后通常不能独立进行生命活动。

### 3. 病毒分类（按寄主）
1. 动物病毒：如流感病毒等。
2. 植物病毒：可引起植物花叶病等。
3. 细菌病毒（噬菌体）：寄生在细菌内。

### 4. 病毒与人类关系
1. 危害
   - 可引发多种传染病。
2. 利用
   - 疫苗研制和免疫预防。
   - 作为分子生物学和基因工程研究工具。
   - 在生物防治中有潜在应用。

---

## 五、本章综合与应用

### 1. 核心比较
1. 细菌 vs 真菌
   - 细菌：原核，单细胞，无成形细胞核，分裂生殖。
   - 真菌：真核，可单细胞或多细胞，多以孢子繁殖。
2. 微生物 vs 病毒
   - 常见微生物（细菌、真菌）有细胞结构。
   - 病毒无细胞结构，必须寄生。

### 2. 与生活实际结合
1. 食物保存方法（低温、干燥、密封）本质是抑制微生物生长繁殖。
2. 饭前便后洗手、规范消毒是阻断病原传播的重要措施。
3. 发酵食品生产需控制温度、时间和卫生条件。

### 3. 高频易错点
1. 把芽孢当作细菌繁殖方式（芽孢主要用于抗逆）。
2. 认为“所有微生物都有害”。
3. 把病毒归为“单细胞生物”。

### 4. 复习建议
1. 画“细菌-真菌-病毒”三者对比表。
2. 结合实验步骤记忆无菌操作和对照实验思想。
3. 用生活案例巩固知识（酸奶制作、食品防霉、传染病预防）。
