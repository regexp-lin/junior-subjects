# 七年级下册生物 第一章 详细知识点

## 一、第一节：种子的萌发
1. 种子由种皮和胚组成，胚是新植物体的幼体。
2. 单子叶植物和双子叶植物种子结构有差异，但都有胚。
3. 种子萌发的环境条件：适量水分、适宜温度、充足空气。
4. 种子萌发的自身条件：胚完整并有生命力，且不在休眠期。
5. 萌发过程中，胚根首先发育成根，随后胚芽发育成茎和叶。

## 二、第二节：植株的生长
1. 根生长一方面依靠分生区细胞分裂增加数量，另一方面依靠伸长区细胞伸长增大长度。
2. 芽是未展开的枝或花，枝条由芽发育而来。
3. 植株生长需要水、无机盐和有机物。
4. 无机盐中含氮、磷、钾对植物生长尤为重要：
   - 含氮：促进枝叶生长
   - 含磷：促进根系发育和开花结果
   - 含钾：促进茎秆健壮、增强抗性

## 三、第三节：开花和结果
1. 花的主要结构是雄蕊和雌蕊，与生殖直接相关。
2. 传粉是花粉从花药落到雌蕊柱头上的过程。
3. 受精是精子与卵细胞结合形成受精卵的过程。
4. 受精完成后：
   - 子房发育成果实
   - 胚珠发育成种子
   - 子房壁发育成果皮
5. 果实和种子的形成是被子植物繁殖的关键环节。

## 四、本章易错点
1. 把“有水就能萌发”当正确（还需温度和空气等条件）。
2. 混淆“子房发育成果实”与“胚珠发育成果实”。
3. 混淆分生区和伸长区在根生长中的作用。
