# 七年级下册生物 第七章 详细知识点

## 一、人体内废物与排泄
1. 细胞生命活动会产生代谢废物，如二氧化碳、尿素、多余水和无机盐。
2. 排泄是将代谢终产物排出体外的过程，不同于排遗（排出食物残渣）。
3. 人体主要排泄途径：
   - 泌尿系统排尿
   - 呼吸系统排出二氧化碳和水
   - 皮肤通过汗液排出部分水和无机盐

## 二、泌尿系统组成与功能
1. 泌尿系统由肾脏、输尿管、膀胱、尿道组成。
2. 肾脏是形成尿液的主要器官。
3. 输尿管将尿液由肾脏输送到膀胱。
4. 膀胱暂时贮存尿液。
5. 尿道负责排出尿液。

## 三、尿液的形成过程
1. 肾脏结构和功能基本单位是肾单位。
2. 肾单位主要包括肾小球、肾小囊、肾小管。
3. 尿液形成的两个关键过程：
   - 过滤作用：血液流经肾小球时，部分物质进入肾小囊形成原尿。
   - 重吸收作用：肾小管把原尿中的全部葡萄糖、大部分水和部分无机盐重新吸收回血液。
4. 最终形成的尿液主要含水、无机盐、尿素等。

## 四、排泄的意义
1. 排出代谢废物，防止有害物质积累。
2. 调节体内水和无机盐平衡。
3. 维持内环境相对稳定。

## 五、健康生活建议
1. 适量饮水，促进代谢废物排出。
2. 不憋尿，降低泌尿系统疾病风险。
3. 注意个人卫生，预防尿路感染。
4. 出现尿频、尿痛、血尿等症状要及时就医。

## 六、本章易错点
1. 把“排遗”当成“排泄”。
2. 认为尿液中应有大量葡萄糖（正常情况下不应有）。
3. 混淆肾脏、输尿管、膀胱、尿道功能分工。
