# 七年级下册生物 第二章 详细知识点

## 一、第一节：水的利用与散失
1. 水是植物体的重要组成成分，也是多种生理活动的介质。
2. 根尖成熟区有大量根毛，是吸收水和无机盐的主要部位。
3. 吸收来的水分经导管由根向茎、叶运输。
4. 蒸腾作用是植物体内水分以水蒸气形式从叶片气孔散失到大气中的过程。
5. 蒸腾作用意义：
   - 促进根吸水和无机盐运输
   - 降低叶片温度
   - 促进水循环
6. 影响蒸腾作用强弱的因素包括光照、温度、湿度、风速等。

## 二、第二节：光合作用
1. 绿色植物在光下利用二氧化碳和水制造有机物并释放氧气。
2. 光合作用主要场所在叶绿体。
3. 叶片结构与光合作用关系：
   - 叶肉细胞含较多叶绿体
   - 气孔有利于气体交换
4. 光合作用公式（理解）：二氧化碳 + 水 在光照和叶绿体条件下 生成有机物 + 氧气。
5. 光合作用意义：
   - 为生物圈提供有机物和能量来源
   - 维持大气中氧气和二氧化碳相对平衡

## 三、第三节：呼吸作用
1. 呼吸作用是细胞利用氧分解有机物，释放能量并产生二氧化碳和水的过程。
2. 呼吸作用在所有活细胞中持续进行，白天和夜晚都发生。
3. 呼吸作用释放的能量用于植物生长、运输和合成等生命活动。
4. 贮藏农产品时需控制温度、湿度和氧气条件以减弱呼吸作用，减少有机物消耗。

## 四、第四节：植物在自然界中的作用
1. 植物通过光合作用固定太阳能，形成生态系统中的生产者。
2. 植物为动物和人类提供食物、纤维、木材和药材等资源。
3. 植物对环境有调节作用：固碳释氧、保持水土、改善小气候。
4. 植被破坏会导致生态失衡，因此应保护森林和草地资源。

## 五、本章易错点
1. 误把“植物只在白天呼吸”当正确。
2. 误把“蒸腾作用无意义”当正确。
3. 混淆光合作用与呼吸作用的原料和产物。
