# 八年级上册生物 第一章 思维导图（树状图）

- 第一章（按 prompt 对应 `ba-shang_7-37.pdf`）
  - 动物的主要类群
    - 一、无脊椎动物
      - 共同特征：体内无脊柱
      - 主要类群
        - 腔肠动物（刺胞动物）
        - 扁形动物
        - 线形动物
        - 环节动物
        - 软体动物
        - 节肢动物
      - 代表与适应特征
        - 水螅、涡虫、蚯蚓、河蚌、蝗虫等
    - 二、脊椎动物
      - 鱼类
        - 鳃呼吸、鳍游泳、适于水生
      - 两栖动物
        - 幼体鳃呼吸，成体肺呼吸并用皮肤辅助
      - 爬行动物
        - 角质鳞片、肺呼吸、陆地产卵
      - 鸟类
        - 羽毛、翼、恒温、气囊辅助呼吸
      - 哺乳动物
        - 被毛、胎生哺乳、恒温、神经系统发达
    - 三、类群比较主线
      - 结构由简单到复杂
      - 生活环境由水生向陆生拓展
      - 生理功能与环境适应不断增强
