# 八年级上册生物 第四章 思维导图（树状图）

- 第四章（按 prompt 对应 `ba-shang_105-123.pdf`）
  - 生物多样性及其保护
    - 一、生物多样性的内涵
      - 1. 遗传（基因）多样性
        - 同一物种内不同个体存在差异
        - 基因多样性是选育良种和适应环境变化的基础
      - 2. 物种多样性
        - 生物种类丰富程度
        - 不同地区物种数有差异（受气候、地形、历史等影响）
      - 3. 生态系统多样性
        - 森林、草原、湿地、海洋等多种生态系统
        - 不同生态系统提供不同生态服务
    - 二、生物多样性的价值
      - 1. 直接价值
        - 食物、药物、工业原料、观赏资源
      - 2. 间接价值（生态价值）
        - 维持生态平衡
        - 涵养水源、防风固沙、净化环境
      - 3. 潜在价值
        - 尚未被发现或利用的遗传资源与药用资源
    - 三、生物多样性面临的威胁
      - 1. 栖息地破坏（开发、污染等）
      - 2. 过度捕捞、滥砍滥伐、乱捕滥猎
      - 3. 外来物种入侵
      - 4. 环境污染与气候变化
      - 5. 违法交易野生生物资源
    - 四、保护生物多样性的措施
      - 1. 就地保护
        - 建立自然保护区、国家公园
        - 保护典型生态系统与濒危物种栖息地
      - 2. 迁地保护
        - 动物园、植物园、种质库、基因库
      - 3. 法律保护
        - 保护野生动植物相关法律法规
        - 打击非法猎捕和交易
      - 4. 科学研究与公众参与
        - 监测、科普、生态修复
        - 绿色消费与低碳生活
    - 五、人与自然关系
      - 生物多样性是人类生存与发展的基础
      - 保护生物多样性就是保护人类自身
