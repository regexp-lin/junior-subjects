## 二、原文及逐段翻译

### 第一段（发现桃花林）

> 晋太元中，武陵人捕鱼为业。缘溪行，忘路之远近。忽逢桃花林，夹岸数百步，中无杂树，芳草鲜美，落英缤纷。渔人甚异之，复前行，欲穷其林。

**译文：** 东晋太元年间，武陵郡有个以捕鱼为职业的人。（一天他）沿着溪水划船，忘记了路程的远近。忽然遇到一片桃花林，（桃树）夹着溪流两岸，长达数百步，（林）中没有其他杂树，芳香的青草鲜嫩美丽，落花纷纷飘洒。渔人对此感到非常惊异，继续往前走，想要走到这片桃花林的尽头。

### 第二段（进入桃花源）

> 林尽水源，便得一山，山有小口，仿佛若有光。便舍船，从口入。初极狭，才通人。复行数十步，豁然开朗。土地平旷，屋舍俨然，有良田、美池、桑竹之属。阡陌交通，鸡犬相闻。其中往来种作，男女衣着，悉如外人。黄发垂髫，并怡然自乐。

**译文：** 桃林在溪水发源的地方就到了尽头，（渔人在那里）便看到一座山，山上有个小洞口，隐隐约约好像有光亮。（渔人）就舍弃了船，从洞口进入。起初洞口很狭窄，仅仅能容一个人通过。又走了几十步，突然变得开阔明亮。（这里）土地平坦宽广，房屋整齐，有肥沃的田地、美丽的池塘、桑树竹林之类的。田间小路交错相通，（村落间）能听到鸡鸣狗叫声。其中来来往往耕种劳作的人，男男女女的穿着打扮，都和桃花源外面的人一样。老人和小孩都安闲快乐，自得其乐。

### 第三段（做客桃花源）

> 见渔人，乃大惊，问所从来。具答之。便要还家，设酒杀鸡作食。村中闻有此人，咸来问讯。自云先世避秦时乱，率妻子邑人来此绝境，不复出焉，遂与外人间隔。问今是何世，乃不知有汉，无论魏晋。此人一一为具言所闻，皆叹惋。余人各复延至其家，皆出酒食。停数日，辞去。此中人语云："不足为外人道也。"

**译文：** （源中人）看到渔人，竟然十分惊讶，问（渔人）从哪里来。（渔人）详细地回答了他们。（他们）就邀请（渔人）到自己家里去，摆酒杀鸡做饭（来款待他）。村里的人听说有这样一个人，都来打听消息。（他们）自己说他们的祖先（为了）躲避秦朝时的战乱，带领妻子儿女和乡邻来到这个与世隔绝的地方，不再从这里出去，于是就与外面的人断绝了来往。（他们）问（渔人）现在是什么朝代，竟然不知道有汉朝，更不用说魏朝和晋朝了。渔人把自己听到的事一一详细地告诉了他们，（他们）都感叹惋惜。其余的人各自又邀请（渔人）到自己家中去，都拿出酒饭（来招待他）。（渔人在这里）住了几天，就告辞离开。这里的人告诉（他）说："（这里的情况）不值得对外面的人说啊。"

### 第四段（离开桃花源与再寻不得）

> 既出，得其船，便扶向路，处处志之。及郡下，诣太守，说如此。太守即遣人随其往，寻向所志，遂迷，不复得路。

**译文：** （渔人）出来以后，找到了他的船，就沿着旧路（回去），到处做了标记。到了郡城，拜见太守，说了这番经历。太守立即派人跟随他前去，寻找先前所做的标记，竟然迷失了（方向），再也没有找到路。

### 第五段（再寻不得）

> 南阳刘子骥，高尚士也，闻之，欣然规往。未果，寻病终，后遂无问津者。

**译文：** 南阳人刘子骥，是个志向高洁的隐士，听到这件事后，高兴地计划前往。没有实现，不久就因病去世了。此后就再也没有探寻（桃花源）的人了。

## 五、内容理解

### （一）文章线索与结构

以**渔人的行踪**为线索：发现桃花源→进入桃花源→做客桃花源→离开桃花源→再寻桃花源。

| 段落 | 内容概括 | 关键词 |
|------|---------|--------|
| 第1段 | 渔人发现桃花林 | 缘溪行、桃花林、异之 |
| 第2段 | 渔人进入桃花源，所见景象 | 豁然开朗、土地平旷、怡然自乐 |
| 第3段 | 渔人做客桃花源，了解来历 | 避秦时乱、不知有汉、不足为外人道 |
| 第4段 | 渔人离开并再寻不得 | 处处志之、遂迷、不复得路 |
| 第5段 | 刘子骥寻访未果 | 未果、寻病终、无问津者 |

### （二）桃花源的特点

| 方面 | 内容 | 原文依据 |
|------|------|---------|
| 自然环境优美 | 土地平坦开阔、物产丰饶 | "土地平旷""有良田、美池、桑竹之属" |
| 社会安定和平 | 没有战乱纷争 | "避秦时乱""不复出焉" |
| 人们生活安乐 | 自给自足、安居乐业 | "往来种作""怡然自乐" |
| 民风淳朴热情 | 热情好客、待人真诚 | "便要还家，设酒杀鸡作食""余人各复延至其家" |
| 与世隔绝 | 不知外界变化 | "乃不知有汉，无论魏晋" |
| 人人平等 | 没有压迫剥削 | 文中未提及任何统治者、税赋 |

### （三）主旨与思想

**中心思想：** 本文通过对桃花源的安宁和乐、自由平等生活的描绘，表现了作者追求美好生活的理想和对当时黑暗现实的不满。

**深层理解：**
- 桃花源是陶渊明虚构的理想社会，寄托了他对**没有剥削、没有压迫、没有战乱、人人安居乐业**的理想世界的向往。
- 文末"寻向所志，遂迷，不复得路"以及"后遂无问津者"，暗示这种理想社会在当时是不可能实现的，具有一定的**批判现实**意义。
- "世外桃源"一词至今仍用来指代脱离现实的安乐美好之地。

## 六、写作手法

| 手法 | 分析 |
|------|------|
| 以渔人行踪为线索 | 贯穿全文，使文章脉络清晰，情节完整 |
| 层层设悬，引人入胜 | 桃花林→山洞→豁然开朗→人物交流→再寻不得，步步推进 |
| 虚实结合 | 桃花源为虚构（理想），以真实的地名（武陵、南阳）、真实人物（刘子骥）衬托，增强真实感与可信度 |
| 详略得当 | 详写桃源内部生活与景象，略写渔人出入经过 |
| 对比 | 桃源内外的对比——源内安宁快乐，源外战乱纷争 |
| 伏笔与照应 | "不足为外人道"与后文"遂迷，不复得路"相照应 |
| 借景抒情 | 通过描写优美和谐的环境，表达对理想社会的向往 |

## 八、中考高频考点总结

### 考点一：重点字词解释
常考词：缘、穷、异、俨然、属、阡陌、交通、妻子、绝境、无论、要、具、咸、悉、并、扶、向、志、诣、规、寻、津、足。

### 考点二：古今异义辨析
常考词：鲜美、交通、妻子、绝境、无论、不足、间隔。

### 考点三：句子翻译
高频翻译句：
1. 芳草鲜美，落英缤纷。
2. 土地平旷，屋舍俨然。
3. 黄发垂髫，并怡然自乐。
4. 自云先世避秦时乱，率妻子邑人来此绝境。
5. 乃不知有汉，无论魏晋。
6. 不足为外人道也。

### 考点四：内容理解
1. 桃花源是一个怎样的社会？（环境优美、生活安乐、民风淳朴、与世隔绝、没有剥削压迫）
2. 文章的主旨是什么？（表达对理想社会的向往和对现实的不满）
3. "不足为外人道也"表现了桃源人怎样的心理？（担忧外界打扰平静生活）
4. 渔人再寻不得说明了什么？（理想社会的虚幻性，美好愿望不可实现）

### 考点五：写作手法赏析
- 虚实结合、对比手法、以渔人行踪为线索、伏笔照应、详略得当。

### 考点六：开放性探究
- 你如何看待陶渊明笔下的"世外桃源"？
- 桃花源的理想对今天有什么启示？
- 陶渊明的思想有何积极意义和局限性？

常与以下篇目对比：
- 《小石潭记》（柳宗元）——借景抒情
- 《岳阳楼记》（范仲淹）——理想社会与忧国思想
- 《醉翁亭记》（欧阳修）——寄情山水
- 《大道之行也》（《礼记》）——理想社会的描述
