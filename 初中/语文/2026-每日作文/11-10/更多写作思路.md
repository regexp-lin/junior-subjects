# 更多写作思路

## 一、多角度切入（不少于 5 个）

### 角度一：对"被忽视的人"的主动善意（推荐）

- **适合写**：公交地铁上的农民工、楼道里的保洁阿姨、校门口卖早点的摊贩、独自坐在食堂角落的陌生同学。
- **可用素材**：周围人的回避态度、"我"的观察与犹豫、主动搭话或帮忙的细节、对方的意外与感激。
- **立意方向**：善良有时是"不忽视"——平等地说一句话，就是对他人最好的尊重。

### 角度二：举手之劳的小善举

- **适合写**：雨天共伞、帮人拎重物、为迷路者指路、电梯里帮人按住开门键、便利店帮人补几块零钱。
- **可用素材**：当时的场景、对方的窘迫或感激、自己"顺手一做"的心态、事后的触动。
- **立意方向**：我们以为微不足道的小事，对他人而言可能是久违的温暖。

### 角度三：善意传递——受过帮助后去帮助别人

- **适合写**：自己曾在陌生环境得到过帮助（如迷路、忘带钱、淋雨），后来在类似情境下主动帮助了别人。
- **可用素材**：当年的窘境与获助、对方的做法、自己后来的仿效、两件事的呼应。
- **立意方向**：善意会传递，每一次被温暖，都可能成为下一次温暖的起点。

### 角度四：从"旁观"到"行动"的转变

- **适合写**：起初看见需要帮助的人时犹豫、不好意思或"别人会帮的"，后来某个契机让自己迈出那一步。
- **可用素材**：内心的纠结、周围人的冷漠或自己的借口、转折点（如想到某句话、某个人）、行动后的释然。
- **立意方向**：善良需要勇气——打破沉默、克服犹豫的那一刻，就是善意的开始。

### 角度五：善意的"意外回报"

- **适合写**：一次无心的小善（如借一支笔、递一张纸巾），后来以某种方式"回到"自己身上，或得到意外的反馈。
- **可用素材**：当时的随意、对方的反应、事后的意外重逢或知晓、对"善有善报"的重新理解。
- **立意方向**：善意不图回报，但善意会留下印记，有时会在不经意间回响。

## 二、叙述方式建议

### 方式一：顺叙（推荐）

- **写法**：按时间顺序写——发现需要帮助的人 → 内心活动 → 决定行动 → 行动过程 → 对方反应 → 事后感悟。
- **优点**：节奏清晰，读者容易跟随，便于铺陈细节和心理变化，符合记叙文常规写法。

### 方式二：倒叙

- **写法**：从对方的一句话或一个画面切入（如"你是今天第一个跟我说话的城里人"），再回溯到当时的场景，交代前因后果。
- **优点**：开篇有冲击力，能迅速抓住阅卷老师，适合那句话特别打动人心的素材。

### 方式三：对比式结构

- **写法**：前半部分写周围人的冷漠或回避，后半部分写自己的行动与对方的反应，形成鲜明对比。
- **优点**：突出"主动善意"的可贵，立意更清晰，但要避免贬低他人，保持客观描写。

## 三、风险提醒

| 风险 | 提醒 |
|------|------|
| **偏题到友情** | 11-04 已有"友情与同行"，本题必须写对**陌生人**或**不熟悉的人**的帮助。同学、朋友的互助不算。 |
| **偏题到换位思考** | 10-29 已有"换位思考与理解"（食堂撞人的宽容），本题强调**主动善意**，而非化解矛盾、理解对方。 |
| **写成空洞说教** | 必须有事例支撑，有具体场景、对话、神态。只喊"善良很重要""要多帮助别人"会显得假大空。 |
| **选材过于老套** | 避免"扶老奶奶过马路""捡钱包还给失主"等陈俗情节。选贴近生活的场景，如公交、电梯、楼道、便利店。 |
| **忽视对方反应** | 善意的分量往往体现在对方的反应上。至少要有一处对方的语言、神态或动作描写，否则容易流于"我做了什么"的流水账。 |
| **助人对象模糊** | 帮助的对象要有具体形象（农民工、保洁阿姨、迷路老人等），不能写成抽象的"需要帮助的人"。 |
