# 肩膀

我对爸爸最早的记忆，跟肩膀有关。

那年我大概四岁，正月十五，爸爸带我去看花灯。人多得像下饺子，我被挤在人群里，什么都看不见，只能看到一片后背和屁股。我急得快哭了，爸爸一把把我捞起来，架在他的肩膀上。

世界一下子变了。

灯笼、人头、远处发光的彩车——所有东西都在我脚底下。我坐在爸爸的肩膀上，两手揪着他的头发，觉得自己是全世界最高的人。爸爸一手扶着我的腿，一手在人群里开路，肩膀稳得像一座山。他的头发被我揪得乱七八糟，他也不管，只是偶尔仰头问我一句："看到了吗？"

"看到了！"

那个晚上，我觉得爸爸的肩膀是世界上最宽的地方。

后来我慢慢长大，不再骑肩膀了。五年级的时候我已经到爸爸的胸口了，六年级到了他的肩膀，初一那年，我发现自己居然可以平视他了。

平视的那一刻，我心里闪过一个奇怪的念头——他是变矮了，还是我长高了？

答案当然是我长高了。但后来发生了一件事，让我觉得，他也许真的变矮了。

初二那年秋天，我们家从老房子搬到新房子。搬家那天请了一辆小货车，大件家具由搬家公司的人搬，零碎的东西我们自己搬。爸爸说他来搬那些纸箱子，让我和妈妈先去新家收拾。

我先走了，走到楼梯拐角的时候回头看了一眼——爸爸正弯腰扛起一个大纸箱，用肩膀顶住箱子的底部，慢慢直起身子。箱子比他的上半身还大，把他的脸遮得严严实实。他走了两步，突然停了一下，换了个姿势，把箱子从右肩换到了左肩。

我看到他右肩的衣服上有一块洇湿的痕迹。不是汗，形状不对。

晚上收拾完新家，妈妈在厨房做饭。我路过浴室门口的时候，门没关紧，从门缝里看到爸爸正对着镜子看自己的右肩——肩膀上贴了一块膏药，膏药边缘发红，皮肤被磨破了一小块。

他一边揭膏药一边轻轻嘶了一声。

我站在门外，喉咙像被什么东西堵住了。

后来我才知道，爸爸的右肩有旧伤。年轻的时候在工地干过活，扛钢管扛伤了肩关节，一到变天就疼。这些年他从来没说过，搬家那天也没说。他只是默默把箱子从右肩换到了左肩，然后继续走。

那天晚上我躺在新房间的床上，想了很多。想起四岁那年骑在他肩上看花灯，想起每次下雨他把伞全遮在我头上自己淋湿了半边肩膀，想起他帮我背书包上楼梯时右手总是不自觉地扶一下右肩——那些我从来没注意过的小动作，此刻一个一个浮了上来。

他的肩膀从来没有真正宽过。它只是在我面前假装很宽。

搬家后的第一个周末，我起了个大早，跑到超市买了两袋米。十斤装的，一袋扛在右肩，一袋夹在左胳膊下面。从超市走回家大概一公里，走到一半的时候肩膀开始酸，米袋硌得锁骨发疼。

到家的时候爸爸正在客厅看电视。他看了一眼我肩上的米袋，又看了一眼我的脸，张了张嘴，最后只说了一句："下次……别买这么多，太沉了。"

我笑了笑："没事，我肩膀宽。"

他没再说话，但嘴角翘了一下。

我知道那一刻他想说什么。他想说的跟当年我骑在他肩膀上时他想的一样——只要你站得高一点、看得远一点，我扛得重一点，没关系。

现在，轮到我了。
