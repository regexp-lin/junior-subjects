# 肩上

父亲的肩膀不宽，甚至有些瘦削，但在我的记忆里，那双肩上似乎永远扛着什么东西——米袋、水桶、我的书包，或者一些我看不见的、沉甸甸的东西。

去年暑假，老家的房子要翻修屋顶，父亲请不起太多工人，决定自己上阵。那天我跟着他去建材店拉瓦片，一箱瓦片少说也有四五十斤，码在三轮车上摇摇晃晃地拉回来。到了家门口，三轮车进不了窄巷子，只能一箱箱往里搬。

父亲弯腰抱起一箱瓦片，扛上右肩，膝盖微微一弯又直起来，深吸一口气，踩着碎石子朝巷子里走去。我看着他的背影——汗水把灰色的T恤洇成了深色的地图，肩膀处的布料被压得紧贴皮肤，能隐约看到肩胛骨的轮廓。他每走一步，脖子上的青筋就凸一下。

"爸，我帮你搬吧。"我跑过去伸手。

"你搬不动，在旁边待着。"他头也不回。

我不服气，弯腰抱起一箱，学着他的样子往肩上扛。刚一上肩，一股钝痛立刻从锁骨传到整条手臂，那重量像是要把我按进地里。我咬着牙迈了几步，碎石子硌着脚底，膝盖发软，肩膀上的瓦片箱每一次颠动都像锤子在砸。走了不到十米，我就觉得呼吸跟不上了，眼前有些发花。

"放下放下！"父亲折回来，一把接过我肩上的箱子，轻巧地挪到自己另一边肩膀上。他两个肩膀各扛一箱，走得稳稳当当，只是步子比刚才慢了些。

"不是叫你别搬吗？你这肩膀还嫩着呢。"他的语气像在责怪，但嘴角好像弯了一下。

那天下午，我没再逞强去扛瓦片。我坐在巷子口，看父亲一趟趟地来回。每一趟经过我身边时，我都能听到他粗重的喘息和瓦片在肩上轻微的晃动声。有一趟他停下来喝水，我看到他右肩的T恤领口被磨破了一小块，里面的皮肤红红的，像被砂纸擦过。他拿矿泉水瓶盖往肩上浇了点水，嘶了一声，然后拧上瓶盖，又弯腰抱起下一箱。

那个瞬间，一种说不清的东西堵在我喉咙里。我突然想起很多以前没在意的画面：小时候他把我扛在肩头逛庙会，我揪着他的头发笑，他的肩膀是我的全世界；下雨天他一只手撑伞一只手拎着我的书包，伞歪向我这边，他的左肩全淋湿了；每次过年回老家，他从车上卸行李，大包小包全挂在两个肩膀上，像一棵挂满了果实的树。

他的肩膀一直在扛东西。扛我长大，扛这个家，扛那些他从不跟我说的辛苦。

搬完最后一箱瓦片，父亲一屁股坐在门槛上，两个肩膀一高一低地耷拉着。我走过去，把矿泉水递给他，然后把手搭在他的肩上。他的肩膀是热的，汗津津的，骨头硌手，肌肉又紧又硬。他抬头看了我一眼，没说话，但我觉得他懂了。

从那天起，每次看到家里有重东西要搬，我都抢着去扛。不是因为我的肩膀已经够硬了，而是我知道——他肩上的东西，早该有人帮他分担一些了。

那些我们扛在肩上的东西，一开始叫重量，后来叫责任。
