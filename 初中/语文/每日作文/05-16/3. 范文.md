# 第一次登台演讲

名字被念到的那一刻，我的大脑一片空白。

"下面有请初三（五）班的林可同学，进行国旗下讲话。"

教导主任的声音从广播喇叭里传出来，回荡在整个操场上空。我站在队列里，脚底像被钉子钉住了一样，身体无法动弹。身后的同学推了我一把："快去啊！"我这才像被按了启动键，机械地迈开步子，穿过队列，走向升旗台。

从队列到升旗台大概五十步。那五十步是我人生中最长的一段路。

周围是一千二百多个人。我没数过，但学校公告栏上写过全校师生人数。一千二百多双眼睛同时看着我，像一千二百多盏聚光灯。我能感觉到自己的耳朵在发烫，脖子上的汗顺着衣领往下淌。手里的演讲稿被我攥成了一个皱巴巴的纸团——我甚至不记得什么时候攥的。

站上升旗台的那一刻，我做的第一件事是低头把稿子展开，试图抚平上面的褶皱。但手指在抖，纸面上的字一会儿清晰一会儿模糊，像水里的倒影。

话筒就在嘴边十厘米的位置。我张了张嘴，没有声音出来。

台下安静得吓人。一千二百多人同时安静的感觉，比一千二百多人同时喧哗还要可怕。

我闭了一下眼睛。

脑海里突然闪过昨天晚上的画面——我站在家里的穿衣镜前，对着镜子练了二十遍开场白。妈妈坐在旁边的椅子上听，每一遍都认真地给我鼓掌。练到第十五遍的时候我烦了，说"算了我不想去了"。妈妈没说什么大道理，只是递了一杯水给我，说了一句："你都练了十五遍了，再练五遍就满了。"

再练五遍就满了。这句话莫名其妙地给了我力量。

我重新睁开眼睛，深吸一口气，把稿子往话筒前面凑了凑。

"老师们、同学们，大家早上好。"

声音从喇叭里出来的时候，我被自己吓了一跳——那个声音又薄又尖，像是别人的。但它确实是从我嘴里发出来的。

第一句话说完之后，第二句就顺畅了一点。第三句，我试着把目光从稿子上抬起来，看了一眼台下。第一排是校领导，表情严肃但不可怕。第二排是初一的学弟学妹，有几个在偷偷聊天。第三排……我看到了我们班的方向，阿杰正对着我比了一个"加油"的手势，夸张得像在做广播体操。

我差点笑出来。但这个"差点笑出来"让我的身体突然松了——肩膀不再耸着，呼吸不再急促，声音也从"又薄又尖"变成了正常的音调。

接下来的三分钟，我不敢说讲得有多好。中间有一处忘词，停顿了大概两秒钟，但我没有慌——我低头瞄了一眼稿子，找到了位置，接着往下讲。还有一处读错了一个字，把"契机"的"契"读成了"楔"，但好像也没人注意到。

最后一句话说完的时候，台下响起了掌声。不是那种礼貌性的稀稀拉拉的掌声，而是真的在拍手。我甚至听到了阿杰的吹口哨声，虽然他的口哨一向吹得像漏气。

走下升旗台的时候，我的腿还是软的。但跟走上去时不同——上去的时候软是因为害怕，下来的时候软是因为释放。

回到队列里，班主任拍了拍我的肩膀，什么都没说，只是笑了一下。

那天晚上写日记的时候，我在本子上画了一个升旗台，旁边写了一行字："一千二百个人看着你的时候，最重要的不是你说了什么，而是你站在了那里。"

第一次的意义，从来不在于完美，而在于——你迈出了那一步。
