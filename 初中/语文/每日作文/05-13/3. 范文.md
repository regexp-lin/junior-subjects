# 藏在画笔里的梦

抽屉最里面那个铁盒子，是我不轻易打开的。

里面没有什么值钱的东西——几支用得只剩笔头的蜡笔、一截折断的2B铅笔、两支秃了毛的水彩笔，还有一支被我咬出牙印的自动铅笔。它们挤在一起，笔身上沾满了干涸的颜料，像一群退休的老兵。

但每一支，都替我记着一段关于画画的日子。

最早的那支蜡笔是蓝色的，只剩下拇指盖那么长。那是我五岁的时候，外婆给我买了一盒十二色蜡笔。我蹲在她家的水泥地上，画了一整个下午。画的什么我已经记不清了，只记得外婆拿起那张纸端详了很久，然后说了一句话："这孩子画得真好，以后能当画家。"

五岁的我当然不知道"画家"是什么意思。但"画得真好"这四个字，像一粒种子落在了心底。

小学三年级的时候，美术老师让我们画"我的家"。别的同学画的都是方方正正的房子，屋顶是三角形，门前有棵树。我画了一条巷子——我家住的那条巷子，两边是高低不齐的旧楼，墙上爬着青苔，巷口有个卖肠粉的阿姨，推着一辆生锈的小推车。老师看了之后把我的画贴在了教室后面的展示墙上，旁边写了一句："观察力很好，有自己的视角。"

那是我第一次因为画画被"看见"。从那天起，我开始认真对待手里的笔。妈妈给我报了周末的素描班，我每个星期六背着画板坐四十分钟公交车去上课。素描班的老师很严格，第一堂课就让我画石膏球体，我画了整整两个小时，橡皮擦碎了一地，线条还是歪的。

"别急，先学会看，再学会画。"老师站在我身后说。

我不太懂什么是"学会看"。但我开始试着观察——观察苹果在灯光下的明暗变化，观察杯子边缘的弧线怎么转折，观察窗帘褶皱形成的阴影。渐渐地，世界在我眼里变得不一样了：不再只是"一棵树""一栋楼"，而是光影、线条、色块的组合。

初一那年，学校举办校园绘画比赛，主题是"我眼中的广州"。我画了一幅水彩——清晨的荔枝湾涌，水面上浮着几片落叶，远处是灰蒙蒙的骑楼，一个穿白色汗衫的老伯正在桥上遛鸟。这幅画拿了二等奖。颁奖的时候，美术老师在台上说："这幅画最难得的不是技巧，而是画面里那股烟火气。"

可到了初二，画画这件事开始变得艰难。课业越来越重，周末素描班的时间和数学补习班撞了，妈妈说："画画又不能考高中，还是先把成绩抓上去。"我没有反驳，因为她说得也没错。素描班停了之后，我的画板靠在角落里积灰，水彩颜料干裂成一片片龟甲纹。

有将近半年，我没碰过画笔。

直到那个周日的午后。我在书柜里翻找参考书，手指碰到了那个铁盒子。打开的瞬间，蜡笔的气味扑面而来——一种混合着蜡油和纸张的、温温的味道。我把那支蓝色蜡笔拿出来，在草稿纸上随手画了几条线。

线条歪歪扭扭的，但手指的感觉回来了。那种笔尖接触纸面时微微的摩擦感，那种脑海里有一个画面正在一点一点成形的感觉——它们从没有真正消失过，只是在铁盒子里安静地等着我。

我重新拿起了画笔。不再去素描班了，就在家里画。每天做完作业后画半小时，有时候画窗外的天空，有时候画桌上的杯子，有时候什么都不想，只是让笔在纸上走。

梦想不一定要大声说出来。它可以很安静，安静地藏在一支咬出牙印的铅笔里，藏在一盒干裂的颜料里，藏在每一张画过的纸的纹路里。只要你还在画，它就还活着。
