# 更多写作思路

## 一、多角度切入（不少于 5 个可写角度）

### 角度一：自己说出实话——承认错误

**适合写**：曾隐瞒过的事情（如弄坏了别人的东西、考试作弊、说了谎）某次鼓起勇气坦白。

**可用素材**：向同学承认弄丢了他的书、向父母坦白成绩、向老师承认抄了作业。

**立意方向**：实话让人心安，真诚才能赢得原谅与信任。

### 角度二：他人对自己说的实话——批评或提醒

**适合写**：朋友、老师或家人当面指出自己的缺点或问题，起初不舒服，后来感激。

**可用素材**：同学说你太爱面子、老师说你的方法不对、父母说你有时候太自私。

**立意方向**：实话有时刺耳，却是最难得的真诚；能对你说实话的人，是真心为你好。

### 角度三：长辈一句朴实的实话

**适合写**：爷爷奶奶、父母或老师说的没有修饰的真心话，让你印象很深。

**可用素材**：爷爷说"做人要实在"、妈妈说"我不怕你考不好，怕你骗自己"、老师说"你其实可以更好"。

**立意方向**：朴素的实话里，藏着最深的爱和期待。

### 角度四：因实话重建的信任

**适合写**：两人之间曾有猜疑或隔阂，因为一方或双方说出真相，关系得以修复。

**可用素材**：好友坦白当初疏远的原因、父母向孩子解释家里的难处、自己向朋友说明误会的来龙去脉。

**立意方向**：实话是信任的起点，坦诚才能让关系走得更远。

### 角度五：目睹不公时的那句实话

**适合写**：看到不公平的事（如有人被冤枉、有人作弊、有人被欺负），自己站出来说出真相。

**可用素材**：为被冤枉的同学作证、指出插队的人、说出自己看到的真实经过。

**立意方向**：实话需要勇气，但值得；正义有时就藏在那一句真话里。

## 二、叙述方式建议

### 方式一：倒叙

从"那句实话"造成的当下结果写起，再回溯到说出的情境。如：先写两人如今又是无话不谈的好友，再写"这一切，要从她那句实话说起"。

**好处**：制造悬念，吸引读者；便于突出实话带来的改变。

### 方式二：插叙

以回忆或联想的方式，在叙述主线中插入与"实话"相关的过往片段。如：某次类似的场景，触发对当年那句实话的回想。

**好处**：打破平铺直叙，增加层次感；便于对比"以前"与"现在"。

### 方式三：双线交织（进阶）

一条线写"那句实话"出现的情境，另一条线写自己之前的误解或隐瞒，两条线在实话说出时交汇。

**注意**：初中阶段慎用，需把握好篇幅与节奏，避免结构过于复杂。

## 三、风险提醒

| 风险 | 应对建议 |
|------|----------|
| **偏题到"谎言"** | 全文重心必须放在"那句实话"上，谎言只可作对比或背景，不可喧宾夺主 |
| **空洞喊口号** | 必须用具体事件支撑，写出谁在什么时候说了什么、当时的情境和反应 |
| **"那句"不够明确** | 文中应出现或点明那句话的大致内容，让阅卷者清楚"那句"指什么 |
| **情感失真** | 避免过度戏剧化，保持初中生口吻，写出真实可感的心理变化 |
| **立意过于泛化** | 不要写成"诚实很重要"的空泛议论，要紧扣"那一句"和具体的人与事 |

## 四、快速自检清单

- [ ] 是否明确了"那句"具体指什么？
- [ ] 是否有至少 2 处细节描写（心理/动作/神态/环境）？
- [ ] 情感是否有变化、有层次？
- [ ] 结尾是否有适度升华，而非仓促收尾？
- [ ] 是否避免了空泛说教和假大空表达？
