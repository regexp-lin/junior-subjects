# 礼物

搬家那天，我从旧书桌最底层的抽屉里翻出了一个铁盒。

铁盒不大，巴掌那么宽，盖子上锈迹斑斑，印着一朵褪了色的牡丹花。我不记得见过这东西。

"妈，这是什么？"我举着铁盒问。

妈妈从纸箱后面探出头，看了一眼，忽然笑了。那个笑容里有一种说不清的柔软。

"打开看看。"

我掀开盖子。里面不是什么贵重物品——

一颗发黄的小牙齿，用纸巾仔细包着。旁边用圆珠笔写了一行歪歪扭扭的字："2018年3月12日，乐乐掉了第一颗牙。"

一张皱巴巴的画，蜡笔涂得乱七八糟，依稀能看出是一个大人牵着一个小人，头顶画了个圆圆的太阳。画的背面写着："幼儿园大班 母亲节礼物。"

一张小学三年级的语文单元测试卷，98分。试卷最上方有一行红色的批注："作文写得非常棒！"

一根红色的头绳，已经完全失去了弹性。

一张大头贴——我缺了门牙咧嘴笑着，妈妈的头发还是长长的，扎着马尾辫。

铁盒不大，东西不多，可我蹲在地上看了很久很久。手指一件件轻轻摸过去，每一样东西都带着温度——不是触觉上的温度，而是某种穿越了十几年光阴的温热，像冬天手心里攥了很久的暖水袋。

"你小时候掉第一颗牙的时候，哭了整整十分钟。"妈妈走过来，在我旁边蹲下，声音很轻，"我哄了半天，说牙仙子会来收走的，你才不哭了。后来我就把那颗牙收起来了。"

"那这张画呢？"

"母亲节你画了送给我的。你说大的那个人是我，小的那个是你，你要一直牵着我的手。"她笑了一下，伸手把画翻过来，"老师让每个小朋友画一幅画送妈妈。你们班三十个孩子，只有你画的太阳是方的。"

我也忍不住笑了。可笑着笑着，鼻子就酸了。

不是因为那颗牙或那张画——而是因为这个铁盒本身。这些东西对我来说只是早已模糊的记忆碎片，可妈妈一件不落地收着，整整收了十几年。那颗小牙齿，我早就不记得了，她还用纸巾仔细地包好，连掉落的日期都标上了。那张涂得歪歪扭扭的画，任何人看了都会觉得一团糟，可她叠好放进铁盒的时候，大概觉得那是世界上最好的画。

我忽然意识到一件事。这些年来我一直以为"礼物"是生日蛋糕上的蜡烛、新年红包里的钞票、考了好成绩之后的奖品。可这个铁盒告诉我，最珍贵的礼物从来不用花钱买。它是一个人愿意把你的成长一点一点收藏起来，像收藏宝贝一样小心翼翼。

那天晚上，我在新家的书桌上放好了那个铁盒。它是妈妈给我的礼物——不，准确地说，里面每一件东西都是我成长的痕迹，是妈妈从岁月里一点一点拾起来、用心保存的。而那份"舍不得丢掉"的心意本身，才是她送给我最好的礼物。

搬家丢了很多东西，但这个铁盒，我会一直带着。
