# 既然来了

我是被"志愿时长加分"骗去的。

暑假社区组织清理河道边的垃圾，招十个志愿者，参加的人可以在综合素质评价里加两个学时。我妈帮我报了名，我没怎么反对——两个学时，半天搞定，划算。

集合那天早上八点，太阳已经很毒了。社区王叔发了垃圾袋、长夹子和一次性手套，指了指河道两岸大约三百米长的范围，说："把能看见的垃圾都捡干净，包括草丛里的。"

我领了工具，心想三百米而已，十个人分一分，一人三十米，半小时就完事了。

结果走到河边我才知道什么叫"能看见的垃圾"。河道两岸的草丛里塞满了东西——塑料瓶、泡沫饭盒、烂掉的塑料袋、烟头、甚至还有一只拖鞋。草有半人高，要弯下腰把草拨开才能看见底下藏了什么。太阳晒在后背上像一块烫铁，汗顺着脖子往下淌，手套里也全是汗，黏糊糊的。

不到二十分钟，我就开始烦了。

我开始只捡大件——塑料瓶好拿，泡沫盒好夹，至于草丛深处那些碎小的烟头和纸屑，我选择假装没看见。旁边几个同学也差不多，有人开始掏手机，有人蹲在树荫下说"等一会儿再干"。

我正想找个借口也歇一会儿，余光瞥见刘阿姨。

刘阿姨是社区的清洁工，五十多岁，今天也来帮忙。她没穿什么志愿者马甲，就穿着自己那件洗得发白的蓝色工作服，戴着一顶草帽。她蹲在河岸最远的那一段，一只手拨开草丛，另一只手拿夹子，一个烟头一个烟头地往外捡。她的动作不快，但一直没停过。

我看了她一会儿，发现她的垃圾袋已经装了大半袋，而我的袋子才盖住了底。

不知道为什么，我有点不好意思。

我重新弯下腰，走回那片我刚才跳过去的草丛。拨开草叶，果然底下还有七八个烟头和几片碎纸。我蹲下来，一个一个夹起来。烟头很小，夹子不太好夹，我干脆用手套直接捏着扔进袋子里。

蹲在草丛里的时候，我忽然发现了一些之前没注意到的东西——草根底下有一只蚂蚱在跳，河面上有几只蜻蜓掠过水面，远处的水其实没那么脏，还能看到几条小鱼在石头缝里游。这些东西之前我一直在看手机，根本没注意到。

刘阿姨走过来倒垃圾的时候看见我在蹲着捡，说了一句："小伙子不错，干得仔细。"

"阿姨，您每天都干这种活吗？"我问。

"河道每周清一次，小区每天扫两遍。"她把袋子口扎紧，"干了十几年了，习惯了。"

"不觉得辛苦吗？"

她想了想，说："辛苦是辛苦，但干净了看着舒服。既然干了这行，就把它干好嘛。"

那句话像针一样扎了我一下。

我想到自己来之前的算计——两个学时，半天搞定，划算。我把这件事当成一笔交易，完成了就走。可刘阿姨不是。她不是来加学时的，她就是在干自己的活，每天干，认真干，十几年了。

上午十一点，三百米河道清理完了。我的垃圾袋装了满满两袋，手套已经不能看了。后背的衣服能拧出水来，腿也蹲得发麻。但看着身后那段干干净净的河岸，心里有一种说不出的舒坦——不是"完成任务"的轻松，而是"做好了一件事"的踏实。

回家路上，我把手套扔进垃圾桶，发现手心磨出了一道浅浅的红印。我握了握拳头，笑了一下。

既然来了，就把事情做好。这句话，刘阿姨说的是她自己，但我打算也借来用用。
