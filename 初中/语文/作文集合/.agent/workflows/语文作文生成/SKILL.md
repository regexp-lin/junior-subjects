# 语文作文辅导

给定一篇作文题目，从多维度进行专业分析，在 **作文题目所在的同级目录下** 生成以下三个文件：

| 序号 | 文件名 | 内容 |
|------|--------|------|
| 1 | `范文作文要点.md` | 作文的核心要点，每个要点 1–2 句话总结 |
| 2 | `写作思路.md` | 如何思考这篇作文，从审题到下笔的完整思维过程 |
| 3 | `完整范文.md` | 一篇完整的高质量范文 |

适用于初中和高中阶段。

## 学段要求

| 学段 | 字数要求 | 立意深度 | 语言水平 | 文体偏好 |
|------|---------|---------|---------|---------|
| **初中** | 600–800 字 | 贴近生活、真情实感 | 简洁流畅、适当修辞 | 以记叙文为主 |
| **高中** | 800–1000 字 | 深刻思辨、多角度论证 | 文采飞扬、引经据典 | 议论文 / 记叙文均可 |

> 如用户未指定学段，默认按 **高中** 要求处理。可以在分析前主动询问用户学段。

## Workflow

当用户给出一个作文题目后，按以下步骤 **依次** 执行。

### 1. 确定输出目录

- 如果用户提供了作文题目所在的文件路径，则在该文件的 **同级目录** 下生成三个文件
- 如果用户只是口头给出题目，则询问用户希望将文件保存在哪个目录
- 如果目录下已存在同名文件，询问用户是否覆盖

---

### 2. 审题分析（内部步骤，不单独输出文件）

对题目进行全面审题，作为后续生成三个文件的基础：

#### 2.1 抓关键词

- 找出题目中的核心概念（名词、动词、形容词）
- 标注哪些是限定词（缩小范围），哪些是中心词（确定方向）

#### 2.2 辨题型

判断作文类型并确定应对策略：

| 题型 | 特征 | 应对策略 |
|------|------|---------|
| 命题作文 | 题目即标题 | 精准扣题，不可偏离 |
| 半命题作文 | 需要补全 | 补题即立意，选择最有话说的方向 |
| 材料作文 | 给出文本材料 | 提炼材料核心观点，多角度切入 |
| 话题作文 | 围绕话题自拟标题 | 化大为小，找准切入点 |

#### 2.3 挖隐含义

- 分析题目是否有比喻义、象征义、引申义
- 指出常见的理解误区
- 说明题目的写作边界（什么可以写，什么不该写）

#### 2.4 明确限制

- 检查是否有文体限制、人称限制、字数要求等
- 如果题目有引导语，分析引导语的暗示方向

---

### 3. 立意与选材（内部步骤，不单独输出文件）

#### 3.1 立意构思

确定 **至少 3 个** 不同层次的立意方向，覆盖：

| 层次 | 说明 | 示例（题目"路"） |
|------|------|----------------|
| **个人层面** | 成长、品格、情感 | 人生道路上的选择与坚持 |
| **社会层面** | 责任、担当、时代 | 新时代青年的奋斗之路 |
| **哲理层面** | 辩证、思辨 | 路的本质——走的人多了便成了路 |

选定 **推荐立意**，并说明推荐理由（新颖度、深度、可写性）。

#### 3.2 选材

为推荐的立意方向准备素材：

| 素材类型 | 具体内容 | 使用方式 |
|---------|---------|---------|
| **亲身经历** | 具体的生活场景建议 | 记叙文主体 / 议论文例证 |
| **名人事例** | 具体人名 + 事迹概述 | 议论文论据 |
| **经典名言** | 原文 + 出处 | 开头引用 / 论证支撑 |
| **历史典故** | 典故名 + 简要内容 | 类比论证 / 丰富文采 |
| **时事热点** | 近年新闻事件 | 增加时代感 |
| **诗词积累** | 相关古诗词 | 增添文采 |

> **选材原则**：典型、新鲜、与主题高度吻合。避免老套素材（如爱迪生发明灯泡、司马迁写史记等过度使用的例子）。

---

### 4. 生成文件一：`范文作文要点.md`

使用 `write_to_file` 在目标目录下创建此文件。

将整篇作文浓缩为 **5–8 个核心要点**，每个要点用 **1–2 句话** 总结。要点应覆盖文章从头到尾的关键内容，让学生一看就知道每段该写什么。

文件内容格式：

```markdown
# 作文要点：《{题目}》

> 学段：{初中/高中} | 文体：{记叙文/议论文} | 字数：{要求范围}

1. **[要点标题]**：[1-2句话说明这个要点要写什么内容、传达什么意思]
2. **[要点标题]**：[1-2句话说明]
3. **[要点标题]**：[1-2句话说明]
4. **[要点标题]**：[1-2句话说明]
5. **[要点标题]**：[1-2句话说明]
...
```

要点撰写规则：

- **第1个要点** 对应开头段，说明如何切入主题
- **中间要点** 对应文章主体，每个要点是一个核心论点或关键情节
- **最后一个要点** 对应结尾段，说明如何收束全文、升华主题
- 每个要点必须具体，不能泛泛而谈（如"写一个例子"是不够的，要说明写什么例子、为什么写）
- 要点之间要有逻辑递进或并列关系，体现文章结构

---

### 5. 生成文件二：`写作思路.md`

使用 `write_to_file` 在目标目录下创建此文件。

详细说明 **如何思考这篇作文**，即从拿到题目到下笔的完整思维过程。教学生"怎么想"而不仅是"怎么写"。

文件内容格式：

```markdown
# 写作思路：《{题目}》

> 学段：{初中/高中} | 推荐文体：{记叙文/议论文}

## 第一步：看到题目，你应该想到什么？
[引导学生从题目出发，联想到相关概念、经历、观点]

## 第二步：确定你要表达的核心观点
[帮助学生从多个方向中选定一个立意，说明选择的理由]

## 第三步：想好用什么素材来支撑
[指导学生如何从自己的积累中选择合适的素材]

## 第四步：理清文章的行文逻辑
[说明段落之间的逻辑关系：先写什么、再写什么、为什么这样安排]

## 第五步：想好如何开头和收尾
[提供开头和结尾的思考方向，注意首尾呼应]

## 常见误区提醒
[列出同学们容易犯的 2-3 个错误，以及如何避免]
```

写作思路的撰写规则：

- 用 **对话式语气**，像老师在课堂上引导学生思考一样
- 每一步都要有 **具体的思考方向**，不能只说"好好想想"
- 结合本题目给出 **针对性** 的思路，不要泛泛的套话
- 可以用 **反问句** 引导思考，如"如果换一个角度想呢？"
- 适当提醒 **常见的思维陷阱**，如"很多同学会只写表面意思，但更好的做法是……"

---

### 6. 生成文件三：`完整范文.md`

使用 `write_to_file` 在目标目录下创建此文件。

根据前面的审题、立意、选材分析，生成一篇完整的范文。

文件内容格式：

```markdown
# {范文标题}

{正文内容，按学段要求控制字数}

---

> **点评**
> - 本文立意：...
> - 结构特点：...
> - 语言亮点：...
> - 预估得分（满分60）：XX 分
```

范文要求：

#### 6.1 开头技巧（任选一种或组合）

- 引用名言 / 诗词，自然引出主题
- 设问或反问，引发思考
- 场景描写，营造氛围
- 排比句式，增强气势

#### 6.2 正文要求

- **紧扣主题**：每段都要回扣中心论点或主题
- **详略得当**：重点段落充分展开，次要段落简洁过渡
- **细节描写**（记叙文）：动作、神态、心理、环境等细节让文章有画面感
- **论证有力**（议论文）：事实论据 + 道理论据结合，避免空谈
- **过渡自然**：段与段之间使用过渡句衔接
- **善用修辞**：比喻、排比、拟人等让文章生动

#### 6.3 结尾技巧（任选一种或组合）

- 呼应开头，首尾圆合
- 升华主题，提升境界
- 发出呼吁，引人深思
- 留有余味，意犹未尽

#### 6.4 语言标准

- 初中：语言朴实真挚，句式灵活，适当使用修辞
- 高中：语言有文采，善用古诗词和名言，句式多变

---

### 7. 汇报结果

三个文件全部生成完成后，向用户展示结果摘要：

```
✅ 已在 {目录路径} 下生成以下文件：

| 文件 | 说明 |
|------|------|
| 范文作文要点.md | {要点数量} 个核心要点 |
| 写作思路.md | {步骤数量} 步思维引导 |
| 完整范文.md | {字数} 字，{文体}，预估 {分数} 分 |
```

## Customization

- 如果用户指定了文体（如"写议论文"），直接按指定文体执行，跳过文体选择
- 如果用户给出了自己的立意或素材，优先围绕用户提供的内容展开
- 如果用户需要针对多个题目进行分析，可以逐一分析，每个题目生成一组三个文件
- 如果用户只需要部分文件（如"只要写作思路"），则只生成对应文件
- 如果用户要求修改某个文件的内容，只重新生成该文件

## Important Notes

- 三个文件必须保持一致性：要点、思路、范文应基于相同的立意和选材
- 素材中涉及的名人事例和名言必须准确，不可捏造
- 初中阶段的思路引导应更具体、更贴近生活
- 高中阶段的思路引导应强调思辨性和深度
- 如果题目有明确的价值导向（如弘扬正能量），分析应符合主流价值观
- `写作思路.md` 用对话式语气，让学生感到亲切，像在和老师交流
- 范文字数严格控制在学段要求范围内
- 使用 `write_to_file` 工具创建文件，不要仅在聊天中输出内容
