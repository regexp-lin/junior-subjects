# 中考语文作文每日生成

## 1. 核心目标

每天为中考备考学生生成 **1 篇**高质量的广东中考语文作文训练材料，体现"会审题、会立意、会构思、会表达"的完整训练链路。输出包含从题目到要点、从范文到多角度思路的完整学习闭环。

## 2. 核心要求与命题趋势

### (1) 基本要求

- **字数要求**：范文不少于 600 字，建议 600-800 字
- **文体控制**：以记叙文为主，必须包含"叙事 + 抒情 + 适度议论"
- **质量标准**：达到一类卷标准——中心明确、材料具体、结构完整、情感真切

### (2) 广东中考命题趋势

| 方向 | 说明 |
|------|------|
| **立德树人** | 体现积极向上的价值观（爱国、勤奋、感恩、诚信、责任） |
| **思辨思维** | 引导学生独立思考，体现对生活和社会的理解 |
| **情感真挚** | 表达真实感受，避免空洞说教 |

### (3) 题目类型覆盖

题目应尽量覆盖以下类型，在近 15 天内轮换使用：

| 类型 | 典型方向 | 参考题目风格 |
|------|---------|-------------|
| **成长类** | 挫折与成长、蜕变、突破自我 | "那一次，我真的长大了" |
| **情感类** | 亲情、友情、师生情、感恩 | "温暖的守候" |
| **视角转换类** | 换位思考、理解他人 | "原来，是这样" |
| **半命题** | 需补全题目后写作 | "____让我懂得____" |
| **命题作文** | 直接给定完整题目 | "主角"、"光亮"、"那个人" |

### (4) 选题约束

- **难度适中**：符合初中阶段语文水平，不偏向高中议论文风格
- **价值导向**：优先"真、善、美"导向，避免猎奇和负向表达
- **不重复**：通过扫描近 15 天已生成的作文，确保题目类型和主题方向不与近期重复

## 3. 输出结构与文件规范

每次在当日目录中必须产出以下 **4 个文件**，文件名固定：

### 3.1 `1. 题目.md`

内容要求：
- 当日作文题目（命题或半命题，半命题需展示补全示例）
- 题目类型（命题/半命题/材料引导）
- 选题理由（1-2 句话，说明与中考趋势的对应关系）

格式示例：

```markdown
# 今日作文题目

## 题目

【命题作文】那束光

## 题目类型

命题作文

## 选题理由

契合广东中考"成长+情感"双主线命题趋势，引导学生关注生活中的温暖与力量，体现立德树人导向。
```

### 3.2 `2. 写作要点.md`

内容要求：

**审题分析**
- 关键词提取与题意边界
- 审题误区与避坑提醒

**立意方案**
- 提供 2-3 个立意方向
- 每个方向简要说明适合写什么内容

**结构思路**
- 推荐"起承转合"或"双线叙事"等结构
- 建议段落安排（开头、展开、高潮、结尾）

**选材建议**
- 可用的生活素材类型
- 具体场景举例

**表达注意点**
- 叙事、抒情、议论如何搭配
- 细节描写的要求（至少 2 处心理、动作、神态或环境细节）

### 3.3 `3. 范文.md`

内容要求：
- 输出 1 篇完整高质量范文
- 字数不少于 600 字，建议 600-800 字
- 有具体事件、细节描写、情感变化与结尾升华
- 至少包含 2 处可感知的心理或细节描写
- 保证原创，不套作、不抄袭
- 末尾标注字数，如"（全文约 XXX 字）"

范文质量检测点：
- ✅ 字数达标：不少于 600 字
- ✅ 细节达标：至少 2 处心理、动作、神态或环境细节描写
- ✅ 立意达标：结尾有思辨式总结或价值升华
- ✅ 结构达标：叙事主线清晰，段落过渡自然

### 3.4 `更多写作思路.md`

内容要求：

**多角度切入**
- 围绕同一题目提供不少于 5 个"可写角度"
- 包括：人物角度、时间角度、场景角度、冲突角度、象征意象角度等
- 每个角度给出：适合写什么、可用什么素材、可能的立意方向

**叙述方式建议**
- 至少给出 2 种"不同叙述方式"建议
- 如：倒叙、插叙、双线结构、第一/第三人称切换

**风险提醒**
- 明确各角度的偏题风险
- 避免空泛与假大空的提醒

## 4. 内容质量控制

### 避坑指南

| 禁忌 | 说明 |
|------|------|
| **禁止套作、抄袭** | 内容必须保持原创和新鲜度 |
| **避免空洞抒情** | 观点必须落在具体事件上 |
| **避免偏题** | 写作过程中持续回扣题目关键词 |
| **避免假大空** | 少喊口号，多写真实体验与真实细节 |

### 质量标准

- **中心明确**：立意要聚焦，不空泛
- **材料具体**：有真实可感的事件和细节
- **结构完整**：起承转合或清晰的自然分层
- **情感真切**：避免 AI 腔，像真实优秀学生作文

## 5. 文件生成与保存规则

### Workflow

#### 步骤一：确认日期与输出目录

- 如果用户指定了日期（如"生成 03-08 的作文"），使用指定日期
- 如果用户未指定，使用**当天日期**
- 日期格式化为 `MM-DD`（如 `03-08`）
- 目标目录：`/Volumes/数据盘/Codes/junior-subjects/初中/语文/每日作文/[MM-DD]/`
- 如果该日期目录已存在文件，询问用户是否覆盖

#### 步骤二：扫描近 15 天作文（防重复）

**这一步至关重要，必须在生成作文之前执行。**

扫描 `/Volumes/数据盘/Codes/junior-subjects/初中/语文/每日作文/` 目录下最近 15 天的作文文件：

1. 列出 `每日作文` 目录下所有日期子目录
2. 按日期排序，取最近 15 天的目录
3. 读取每个目录下的 `1. 题目.md` 文件
4. 从每个文件中提取以下信息：
   - **题目类型**（命题/半命题/材料引导）
   - **主题方向**（成长/情感/感恩/挫折/友情 等）
   - **作文题目**
5. 建立**近 15 天已用题目类型+主题列表**

选题规则：
- 新作文的**主题方向**不得与近 15 天的任何一篇重复
- 新作文的**题目类型**尽量避免与最近 3 天的题目类型相同
- 如果近 15 天内某些题目类型或主题方向从未出现过，优先选择这些未覆盖的方向

#### 步骤三：自动选题并生成作文

根据步骤二的去重结果，自动选择一个近 15 天未出现过的题目类型+主题组合，按照第 3 节的输出结构生成完整的作文训练材料。

#### 步骤四：写入本地文件

在目标目录下创建 4 个文件：
- `1. 题目.md`
- `2. 写作要点.md`
- `3. 范文.md`
- `更多写作思路.md`

#### 步骤五：汇报结果

文件生成完成后，向用户展示结果摘要：

```
✅ 已在 每日作文/[MM-DD]/ 目录下生成以下文件：

| 项目 | 内容 |
|------|------|
| 题目类型 | {题目类型} |
| 主题方向 | {主题} |
| 作文题目 | {题目} |
| 范文字数 | 约 {XXX} 字 |
| 文件路径 | {完整路径} |

近 15 天已覆盖主题：{列出近期主题方向}
```

## 7. Customization

- 如果用户指定了日期，使用指定日期而非当天日期
- 如果用户指定了题目类型或主题（如"写一篇关于亲情的命题作文"），按用户要求生成，但仍检查是否与近期重复并提醒
- 如果用户要求生成多篇，逐篇生成，每篇主题不同
- 如果用户要求只输出范文部分，只生成范文相关内容
- 如果用户要求修改某个部分，只重新生成该部分
- 如果用户提供了自己写的作文要求点评，转为点评模式而非生成模式

## 8. Important Notes

- **防重复是核心**：每次生成前必须扫描近 15 天的文件，确保主题方向不重复
- **输出 4 个文件**：题目 → 写作要点 → 范文 → 更多写作思路，体现完整训练链路
- **范文字数严格控制**在 600-800 字，过长过短都不符合中考要求
- **审题立意是重中之重**：精准解码题目，挖掘正向价值导向
- **细节描写必须到位**：至少 2 处心理、动作、神态或环境细节
- **避免假大空**：少喊口号，多写真实体验与真实细节
- **结尾要有升华**：思辨式总结或价值升华
- 使用 `Write` 工具创建文件，不要仅在聊天中输出内容
- 一句话执行原则：**题目像真题，要点能指导，范文可模仿，思路要多元，整体要符合广东中考真实训练场景**
