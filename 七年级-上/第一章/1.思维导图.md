# 七年级上册生物 第一章《认识生物》思维导图（树状图）

- 第一章 认识生物
  - 第一节 观察周边环境中的生物
    - 1. 观察的意义
      - 观察是学习生物学的基本方法
      - 观察不是“随便看看”，而是“仔细察看”
      - 观察是科学探究的基本方法之一
    - 2. 观察前准备
      - 明确观察目的
        - 观察生物的形态、结构特征
        - 观察生物的生活环境
        - 观察生物的生活习性
        - 比较不同生物的共同点与不同点
      - 明确观察对象
        - 校园、公园、林地中的植物和动物
      - 观察工具
        - 肉眼
        - 放大镜、显微镜、望远镜
        - 照相机、摄像机、录音机
        - 必要时进行测量
    - 3. 观察实施要点
      - 观察植物
        - 高度、形态、生长环境
        - 比较树干、枝条、叶、花等特征
        - 注意爱护植物
      - 观察动物
        - 形态特点、运动方式、生活环境
        - 观察鸟类要点
          - 不影响其正常生活
          - 着装不鲜亮
          - 保持安静
          - 拍摄尽量不用闪光灯
      - 野外观察要求
        - 注意安全
        - 分组进行
        - 听从老师安排
    - 4. 观察记录与交流
      - 真实、客观记录
      - 在结果中找规律
      - 记录表核心项目
        - 观察人、班级、天气、地点、时间
        - 生物名称、特征、生活环境
      - 班内交流观察结果
    - 5. 观察中的科学思维
      - 观察要有明确目的
      - 按一定顺序观察
      - 对长期观察要有计划和耐心
      - 观察时积极思考，多问“为什么”
      - 在观察基础上交流与讨论
      - 发现问题、提出问题（比只记结论更重要）
    - 6. 本节迁移
      - 从观察出发，进一步认识“什么是生物”
      - 生物种类包括：植物、动物、真菌、细菌、病毒等

  - 第二节 生物的特征
    - 1. 核心问题
      - 什么是生物？
      - 怎样区分生物和非生物？
      - 生物有哪些共同特征？
    - 2. 生物与非生物辨析
      - 机器人会“动作/交流”但不是生物
      - 生石花外形像石头但属于生物
      - 钟乳石能“长大”但不是生物
      - 判断生命不能只看某一个现象（如会动或会长）
    - 3. 生物的共同特征
      - （1）除病毒外，生物都由细胞构成
        - 细胞是生物体结构和功能的基本单位
      - （2）生物的生活需要营养
        - 植物：吸收水、无机盐、二氧化碳，光合作用制造有机物
        - 动物：以其他生物为食获取营养
      - （3）生物能进行呼吸
        - 多数生物获取氧气、排出二氧化碳
      - （4）生物能对外界刺激作出反应
        - 植物：含羞草受触碰叶片合拢
        - 动物：狮追斑马、斑马逃跑
      - （5）生物能生长、发育和繁殖
        - 由小长大
        - 发育到一定阶段繁殖后代（产卵或产崽等）
      - （6）生物具有遗传和变异
        - 亲子相似（遗传）
        - 个体间有差异（变异）
      - （7）生物还能排出体内废物
    - 4. 本节结论
      - 生物是具有一系列生命共同特征的物体
      - 判断是否为生物，应综合多种特征，不能只看单一现象

  - 章节学习方法总结
    - 先观察现象，再提炼特征
    - 用“比较法”区分相似对象
    - 用“记录-交流-修正”提高观察质量
    - 用“图解法”整理生物共同特征
